## 筋肉量アップとカロリーの科学#3「めっちゃ正しいプロテイン増量法」

「筋肉量アップとカロリーの科学」の続きです！([#1](https://ch.nicovideo.jp/paleo/blomaga/ar2224587),[#2](https://ch.nicovideo.jp/paleo/blomaga/ar2224660))

このシリーズでは、「 **筋肉を増やすための正しいカロリー設定とは？** 」って問題を掘り下げております。筋肉を増やすためには絶対に適切なカロリー摂取が必要なんだけど、現時点では「筋肉を増やしたいならカロリーは大量に摂れ！」や「筋肉を増やすためにはいつも通りのカロリーでいい！」みたいに真逆の見解が出てまして、いったいどうすればいいのかよく分かんないんですよね。

前回は「どんな人がどれくらい食べればいいのか？」というテーマで、タイプ別のカロリー設定について掘り下げてみました。そこで今回は、そこからもう一歩踏み込んで **「追加のカロリーって何から摂るべきなの？」** という問題に取り組んでみましょう。追加のカロリーはプロテインで増やすべきなのか、それとも炭水化物や脂質でいいのか？みたいな問題ですな。

## 「筋肉を増やしたいけど、太りたくはない」は叶うのか？

「筋肉を増やすためには追加のカロリーが必要なのはわかるけど、脂肪も増えそうで怖い……」

この問題に立ち向かうために、前回は「運動レベルのタイプ別」にカロリーの摂取量の調整を考えたわけです。しかし、当然ながらカロリー以外にも大事なポイントはありまして、それが、

- **プロテインによる追加カロリーなら太りにくい！**

って考え方であります。追加でカロリーを摂取する際に、糖質や脂質よりもたんぱく質を中心にしたほうが、体脂肪を必要以上に増やさずに筋肉をメインに増量できるって説ですな。「プロテインは太らない！」みたいな話は、皆さんも一度は耳にしたことがあるかと思いますが、果たしてどこまで正しいんでしょうか？

そこでまず取り上げたいのが、ノバ・サウスイースタン大学のホセ・アントニオ博士のチームが行った有名な研究([R](https://pubmed.ncbi.nlm.nih.gov/27807480/))であります。これは筋トレを長くやってきた人を対象にしたもので、参加者の半数に以下の食事を指示したんだそうな。

- 普段の食事に加えて、 **1日あたり体重1kgあたり4.4gのタンパク質** を摂取してもらう

つまり、体重が70kgの人なら、1日あたり約308gのプロテインを追加したわけで、なかなかイカれた実験でとても良いですね（ほめている）。

実験の期間は8週間で、参加者の摂取カロリーはおよそ1日＋800kcalにも達したとのこと。月で考えると2万4千kcal近い追加ですから、普通に考えたら「めちゃくちゃ太るはず」なんですが、実際の結果はさにあらず。その結果は、

- 脂肪はほとんど増えず、むしろ微減。体重も横ばい。筋肉は微増。

というものだったんですよ。まさに「食べても太らない」を地で行く感じでして、これだけ見ると「カロリー神話が崩壊した？」みたいな気分になってくるわけです。

## プロテインが太りにくい4つの理由

が、よく考えてみれば、これは不思議なことでもなくて、プロテインが太りにくいのには大きく4つの理由があったりします。

### 1．熱発生（Thermic Effect of Food：TEF）が圧倒的に高い

私たちが食べたものは、消化・吸収・代謝される過程でエネルギーを消費するのはご存じのとおり。これは「食事誘発性熱産生（TEF）」と呼ばれるもので、その大きさは栄養素によって全く違うんですね。ざっくりまとめると、

- タンパク質のTEF＝20〜30%
- 炭水化物のTEF＝5〜10%
- 脂質のTEF＝0〜3%

みたいになりまして、つまり100kcal分のプロテインを摂っても、 **最大30kcalはそのまま熱になって消える** わけです。しかも、これは意識的な運動で消費したカロリーではなく、「何もしなくても勝手に燃える」数字なのがうれしいところ。この燃焼効率の高さが、タンパク質を「太りにくい栄養素」にしている最大の理由であります。

### 2．NEAT（非運動性熱産生）が自然に上がる

こちらもおなじみの観点でしょう。NEATは「Non-Exercise Activity Thermogenesis」の略で、 **日常のちょっとした動きで消費するエネルギー** のこと。たとえば、

- 歩く
- 階段をのぼる
- 掃除をする
- 貧乏ゆすりをする
- 表情やジェスチャーを増やす

などなど、私たちが意識せずにやっている活動は、すべてNEATに含まれます。

で、プロテインをしっかり摂って筋肉がついてくると、 **体の「ベース出力」が上がる** ので、こういった日常の動きも活発になるケースが多め。それと同時に、摂取カロリーが増えることで、体は「燃やさなきゃ」と勝手にNEATを増やす傾向もあったりします。一説には、1日に追加で1,000kcalのカロリーを摂取した人は、 **300〜700kcalも勝手にNEATが増加** する傾向があったそうで、  
これが「同じように食べているのに太る人と太らない人がいる」理由でもあります。

### 3．たんぱく質は満腹感が高く、他の栄養素の“つまみ食い”が減る

たんぱく質は、満腹ホルモン（PYY、GLP-1など）を強く刺激する栄養素のひとつ。そのため、プロテインをしっかり摂っていると、自然とお菓子や脂質の多いものを「なんとなく避ける」ようになったりもします。こういった「 **自発的なカロリーコントロール** 」を起こしてくれるのもプロテインの強みっすね。

また、たんぱく質が多い食品ってのは、咀嚼回数も多くなるせいで消化にも時間がかかるため、「胃にしっかり入っている感」が続きやすいんですよ。これは、空腹でつい手を伸ばしてしまう“余計な間食”を減らしてくれる、地味だけど強力な作用です。

### 4．内臓・皮膚・臓器レベルでの“代謝アップ”が起きる

「筋肉を増やす」というと、つい「腕や脚のサイズ」だけに意識がいきがちですが、体の中でもエネルギー消費は起きております。プロテインをたくさん摂ると、体は「こんなに大量のたんぱく質を処理しきれない！」と判断して、

- 肝臓や腎臓の代謝活動を増やす
- 皮膚のターンオーバーを早める
- 消化酵素の生成量を上げる

といった感じで、いろんな対策を打ち始めるんですよ。この“目に見えない代謝コスト”が意外とバカになりませんで、ある研究(R)では、大量のたんぱく質を食べている期間中に、 **内臓や結合組織の代謝コストが有意に上昇** すると報告されております。つまり、プロテインの過剰摂取は、「見える筋肉」だけじゃなく、体全体に“エネルギーを使わせる”指令を出すスイッチにもなるんですね。

ということで、これら4つの要素が複合的に働くことで、プロテインは「摂っても吸収されにくい」「摂ることで他の活動量も上がる」という2段構えで脂肪に変換されにくくなるわけです。なかなか唯一無二の栄養素ですなぁ。

さらにもうひとつ、また別の研究([R](https://www.tandfonline.com/doi/full/10.1186/s12970-015-0100-0))では、 **体重×3.4gの高タンパク摂取でもリコンプ（筋肉増＋脂肪減）が起きた** と報告されております。この実験は平均トレ歴が2.5年以上の男女48名が対象で、全員に週5回の筋トレをしてもらいつつ、そのうち半分には平均で1日に体重1kgあたり2.3gのたんぱく質を取ってもらい、残り半分には1日に体重1kgあたり3.4gのたんぱく質を摂取してもらったとのこと。その上で体組成をチェックしたら、高タンパク群では、筋肉量が有意に増加し、体脂肪は-1.6%減少したというからすごいもんです。

この研究のポイントは、1日体重1kgあたり3.4gものカロリー（374kcalぐらい）を追加したにも関わらず、脂肪が減ったってところでしょう。つまり、カロリーを増やしたのに **太るどころか、むしろ引き締まった** ってことでして、これはたんぱく質を増やしてみるしかない！って気分になるわけです。これは良い結果ですなぁ。

とはいえ、「じゃあ全員プロテインガブ飲みすればいいのか？」というと、それも違うんでご注意ください。たとえば、ルイジアナ州立大学のジョージ・ブレイ博士らが行った代謝病棟研究([R](https://pubmed.ncbi.nlm.nih.gov/22215165/))では、みんなに平均で40％の追加カロリーを与え（平均約954kcal増）、その際にタンパク質の摂取量を5%、15%、25%の3グループにわけたとのこと。

すると、25％のグループは体脂肪が増えなかった……なんてことはなくて、すべてのグループで体脂肪がちゃんと増えていたそうな。まあたんぱく質の摂取量が多いほど筋肉の減少は少なかったんですが、脂肪の蓄積は止められなかったみたいっすね。要するに、同じように追加でタンパク質を摂取させたのに、しっかり体脂肪が増えたって結果でして、前述のアントニオ先生の研究とは異なる結果になってるわけっすね。なぜこのような食い違いが出たのかと言いますと、両者の研究は、被験者に以下のような違いがあるからです。

| 要素 | アントニオ研究 | ブレイ研究 |
| --- | --- | --- |
| 被験者のタイプ | 筋トレ歴が長い | 一般人 |
| 活動環境 | 日常生活 | 代謝病棟（NEATゼロ） |
| 食事の質 | 高タンパク＋通常食 | 管理された食事 |

つまり、追加のプロテインで引き締め効果を得られるのは、がっつり筋トレしてる現実の日常生活をしている人間だけってことですな。高タンパクの効果を得るためには「筋トレ＋NEAT」を組み合わせなきゃいけないわけっすね。

## じゃあ、プロテインをどうやって追加すべきか？問題

では、上記をふまえた上で、ここからは具体的な実践法を見ていきましょう。たんぱく質を増やすためには、具体的に何をすればいいのかってことですね。

## 目的別プロテイン摂取量の設定

まずは、目標によって摂取量を設定します。筋肉を増やすために、どのくらいたんぱく質を摂ればいいのか？　ってのは永遠のテーマみたいなもんですが、最近の研究をもとにすると、目標ごとに最適な摂取量が変わるのが分かってきております。ということで、まずは「目的別」に必要なたんぱく質量の目安を見ていきましょう。

| 目的 | 推奨タンパク質量（g/kg体重） | 解説 |
| --- | --- | --- |
| **筋肉を維持しながら減量** | 2.2〜2.5 | 筋肉の分解を防ぎ、脂肪優先で落とす |
| **リコンプ（脂肪減＋筋肉増）** | 2.6〜3.2 | 筋肥大と脂肪減を同時に狙う特殊戦略 |
| **増量（なるべく脂肪を増やさず筋肉だけ）** | 2.5〜3.0 | 太りたくない増量民向け、最も現実的 |
| **フルバルク（体重増優先）** | 2.0〜2.2 | 追加のカロリーが大きいならここで十分 |

**例** ：体重60kg、体脂肪率15%の人であれば、除脂肪体重（≒筋肉）は約51kg。リコンプ狙いなら、2.8g × 51kg = 約143g/日が目標になる。

## 1日の中での「分割パターン」

たんぱく質の摂取量が決まったら、次は「いつ、どれだけ摂るか？」もチェックしときましょう。まあ実際には、たんぱく質は摂取量が最も大事なので、分割パターンにそこまで気を配る必要もないんですけど、「1回ドカ食い」するのは大変だって人もいるでしょうから、その場合は以下を参考に「こまめに分けて摂取してみる」のもありです。

| タイミング | 目安量 | 例 |
| --- | --- | --- |
| 朝（起床後〜1時間以内） | 0.4g/kg | ホエイ＋卵、またはギリシャヨーグルト |
| 昼（ランチ時） | 0.4g/kg | 鶏胸肉、または鮭＋副菜＋玄米 |
| トレーニング後30分以内 | 0.4〜0.5g/kg | ホエイ＋バナナ、またはプロテインバー |
| 夜（就寝1〜2時間前） | 0.4g/kg | カゼイン、または豆腐＋味噌汁＋ゆで卵 |

**例** ：体重60kg → 0.4g/kg = 24g → 1日約4回に分けて 24g × 4 = 約96g残りは間食やサプリ、夜食で微調整

## プロテインの種類と活用戦略

同じたんぱく質でも、種類やタイミングを工夫することで効率がガラッと変わるのが筋トレの奥深さ。「いつ、どんなプロテインを使えばいいのか？」という疑問に対して、ここではタイプ別・シーン別に使い分ける戦略を見てみます。

| タイミング | 推奨タイプ | 理由 |
| --- | --- | --- |
| 起床後・トレ後 | **ホエイ** （吸収が速い） | 筋分解を防ぎ、筋肉の成長をすばやく刺激 |
| 就寝前・夜 | **カゼイン** （吸収が遅い） | 就寝中の筋肉の成長を安定供給 |
| 日中の間食 | **ミックス、または食材由来** | 満腹感も得られ、習慣化しやすい |
| 外出・移動中 | **プロテインバー** | 持ち運びやすく、咀嚼感ありでNEATの改善にも役立つ |

**※** 大豆・ピープロテインのみで構成するのは、筋肥大効率的にやや劣るかも。ただし、個人的には、最終的な筋肉の成長にはそこまで影響しないと思ってるので、好きなほうを選べばOK。

## 食事全体でのプロテイン強化法

もちろん、プロテインパウダーだけに頼らず、日々の食事でたんぱく質を底上げするのが理想ではあります。ここでは、高たんぱくな食材をうまく取り入れながら、無理なく合計摂取量を増やしていく方法を整理しておきましょう。

### 高タンパクな食材（100gあたり）

| 食材 | タンパク質量 | 補足 |
| --- | --- | --- |
| 鶏胸肉（皮なし） | 約23g | 調理しやすくコスパもよい |
| 鮭（サーモン） | 約22g | オメガ3も同時に摂れる |
| ギリシャヨーグルト（無糖） | 約10g/100g | 低糖質・高カルシウム |
| 卵（全卵） | 約6.5g/個 | 必須アミノ酸が豊富 |
| 木綿豆腐 | 約6.6g/100g | 大豆タンパク＋植物性エストロゲン |
| プロテインバー | 15〜25g/本 | 間食向け、甘味欲のリセットにも最適 |
| 鰹節・ちりめんじゃこ | 30g/100g超 | トッピングに使うのが吉 |

### 食事への組み込み例（1日）

| 食事 | メニュー例 | タンパク質量 |
| --- | --- | --- |
| 朝 | ギリシャヨーグルト＋卵2個＋プロテイン20g | 約40g |
| 昼 | 鶏むね肉150g＋玄米＋味噌汁 | 約40g |
| トレ後 | ホエイプロテイン30g | 約25g |
| 夜 | 鮭150g＋豆腐＋副菜 | 約40g |
| 就寝前 | カゼインプロテイン20g | 約20g |

ということで、ここらへんのガイドラインをチェックしておくと、かなり効率良く筋肉を育てることができるはずであります。まあ、基本的には食事だけで200g以上ものたんぱく質を摂取するのは難しいんで、現実的にはプロテインサプリで1日50〜100gを補うことになるでしょう。

ちなみに、タンパク質を取りすぎると腎臓に悪いなんて説もありますが、 健康な人であれば体重1kgあたり **3.0g以上でも腎機能への悪影響はない** とする研究が複数ありますんで安心してください(R)。もしも心配なら、定期的な健康診断でeGFR・BUN・クレアチニンなどをチェックしましょう。

## まとめ

いろいろ見てきましたが、今回の話をまとめると、

1. プロテインは、 **脂肪になりにくいカロリー源** である
2. 筋トレ＋高タンパク＋NEATがリコンプの王道ルート。
3. **ただし、「動かない人」がプロテインを追加したら普通に太るので注意**

みたいになります。「追加のカロリーは太るから怖い……」と思っている人は、まず「プロテインなら大丈夫かも」ぐらいに考え直してくださいませ。

ってことで、タンパク質が決まったら、次に考えるべきは「脂質と炭水化物のさじ加減」であります。次回は、 **“太りやすい追加カロリー”の避け方と、筋トレ民向けの増量食の組み方** を掘り下げてみましょう。ではまた！

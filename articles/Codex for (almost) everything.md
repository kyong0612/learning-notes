## 概要

OpenAI は、週 300 万人以上の開発者が利用している Codex に対して、ソフトウェア開発ライフサイクル全体をカバーするメジャーアップデートを発表した。今回のアップデートで Codex は単なるコーディングアシスタントから、**コンピュータを操作し、ブラウザを扱い、画像を生成し、過去の作業を記憶し、長期的なタスクを自走できるパートナー** へと進化した。

アップデートの柱は 3 つ：

1. **コーディングを超えた拡張** — Computer use、アプリ内ブラウザ、画像生成、プラグインエコシステム
2. **開発ライフサイクル全体のサポート** — PR レビュー、マルチターミナル、SSH による remote devbox 接続
3. **時間を超えて作業を引き継ぐ** — 会話スレッドの再利用、スケジュール実行、メモリ、プロアクティブな提案

---

## 1. Codex をコーディングの外へ拡張

### Background Computer Use

- Codex は自身のカーソルで **画面を見て、クリックし、タイピングする** ことで、PC 上のあらゆるアプリを操作できるようになった。
- 複数のエージェントが Mac 上で **並列に動作** し、ユーザー自身の作業を邪魔しない。
- 活用例：
  - フロントエンドの変更を反復改善
  - アプリのテスト
  - API を公開していないアプリ上での作業

### アプリ内ブラウザ（In-app Browser）

- Codex アプリに組み込まれたブラウザ上で、**ページに直接コメント** を書いて指示を与えられる。
- 現時点ではフロントエンド開発やゲーム開発で有用。
- 今後は localhost 上の Web アプリを越えて、**ブラウザ全体を Codex が操作できるように拡張予定**。

### 画像生成（gpt-image-1.5）

- Codex は [gpt-image-1.5](https://developers.openai.com/api/docs/models/gpt-image-1.5) を使って画像の生成・反復編集が可能に。
- スクリーンショットとコードを組み合わせることで、同一ワークフロー内で以下を作成できる：
  - プロダクトコンセプトのビジュアル
  - フロントエンドデザイン
  - モックアップ
  - ゲームアセット

### 90 以上の新規プラグイン

スキル、アプリ連携、MCP サーバーを組み合わせた **90+ の追加プラグイン** をリリース。開発者に特に有用とされるものの例：

- **Atlassian Rovo**（JIRA 管理）
- **CircleCI**
- **CodeRabbit**
- **GitLab Issues**
- **Microsoft Suite**
- **Neon by Databricks**
- **Remotion**
- **Render**
- **Superpowers**

## 2. ソフトウェア開発ライフサイクル全体をサポート

Codex アプリに追加された機能：

| 機能 | 説明 |
|------|------|
| GitHub レビューコメント対応 | PR 上のレビューコメントへの返信・対応を支援 |
| マルチターミナルタブ | 複数ターミナルを同時に扱える |
| Remote Devbox への SSH 接続 | **alpha** 版として提供 |
| サイドバーからファイルを直接開く | PDF・スプレッドシート・スライド・ドキュメントのリッチプレビュー対応 |
| サマリーペイン | エージェントのプラン、参照ソース、生成物（artifacts）を一望できる |

これらの改善により、**コード記述 → 出力確認 → 変更レビュー → エージェントとの協働** までを 1 つのワークスペースで高速に行き来できるようになった。

## 3. 時間を超えて作業を引き継ぐ

### Automations の拡張

- **既存の会話スレッドを再利用** し、それまでに積み上げたコンテキストを保持したまま作業を継続できる。
- Codex は将来の作業を **自分自身でスケジューリング** し、自動的に目覚めて長期タスクを継続できる（**数日〜数週間** にまたがるタスクも可能）。
- チームでの利用例：
  - 未マージ PR を最終的に landing させる
  - タスクのフォローアップ
  - Slack / Gmail / Notion など高速に流れるツール上の会話を追跡

### メモリ（Preview）

Codex に **メモリ機能** のプレビューが導入された。

- 過去の経験から有用なコンテキストを記憶：
  - 個人的な好み
  - 過去の修正（corrections）
  - 収集に時間がかかった情報
- これにより、従来は膨大なカスタム指示でしか到達できなかった品質で、今後のタスクをより高速に完了できる。

### プロアクティブな作業提案

- プロジェクト、接続済みプラグイン、メモリからのコンテキストを統合して、Codex が **次に取り組むべき作業を提案**。
- 例：
  - Google Docs 上で対応が必要な未解決コメントを検出
  - Slack / Notion / コードベースから関連コンテキストを収集
  - **優先度付きのアクションリスト** を提示

これにより「作業日の始め方」や「前回の続きをどこから再開するか」が自然にナビゲートされる。

## 4. 提供状況（Availability）

- 本アップデートは本日より、**ChatGPT でサインインしている Codex デスクトップアプリユーザー** に順次ロールアウト開始。
- **コンテキストを理解した提案 / メモリ** などのパーソナライズ機能は、**Enterprise・Edu・EU・UK** ユーザーへ順次展開予定。
- **Computer use** は当初 **macOS のみ** で提供され、EU / UK ユーザーにも順次展開予定。

## 5. 今後の展望（What's next）

- Codex ローンチから 1 年で、開発者の使い方は大きく広がった：
  - コードを書く
  - システムを理解する
  - コンテキストを集める
  - 作業をレビューする
  - バグをデバッグする
  - チームメイトと調整する
  - 長期的な作業を前進させ続ける
- OpenAI のミッションは「AGI が全人類に利益をもたらすこと」、そしてその一部として **「想像できるもの」と「作れるもの」の差を縮める** ことを挙げている。
- 今回のリリースは、Codex を **ソフトウェア開発に関わるツール・ワークフロー・意思決定** にさらに近づけるものであり、今後も継続的な進化が予告されている。

## 重要な結論・示唆

- **Codex は「コーディングアシスタント」から「ソフトウェア開発ライフサイクル全体を伴走するエージェント」へと位置付けを拡張** している。
- **Computer use + In-app browser + 画像生成** の組み合わせにより、API を持たないアプリや視覚的な成果物（UI、ゲーム、モックアップ）への対応力が大幅に強化された。
- **メモリとスケジューリング** により、**1 回限りのタスク実行**ではなく **日・週単位の長期プロジェクト** を任せられるエージェントへと進化している。
- 90+ のプラグインによる **MCP / 外部サービス連携の拡大** は、Codex を実際のチーム運用（JIRA、GitLab、Slack、Notion 等）に組み込みやすくする狙いが明確。

## 制限事項

- **Computer use は macOS のみでローンチ**（Windows・EU・UK は後日）
- **パーソナライズ（提案・メモリ）は Enterprise / Edu / EU / UK に後日提供**
- **Remote devbox への SSH 接続は alpha 品質**
- In-app browser の「ブラウザ全体制御」はまだ将来計画であり、現時点では主に **localhost 上のフロントエンド・ゲーム開発** 向けユースケースに限定される。

## 概要

「筋肉量アップとカロリーの科学」シリーズの第2回。前回は「多くの人には軽い余剰カロリーが必要」という結論に至ったが、今回は「具体的にどれくらい食べればいいのか？」という疑問に対し、**個人の体質やライフスタイルによって適正なカロリー量を5つのタイプに分類**してガイドする。

## タイプ1：初心者＋体脂肪多め（リコンプ最強ゾーン）

### 該当条件

- 筋トレ歴が浅い（1年未満）
- 体脂肪率が高い（男性15〜25%、女性25〜35%）

### 特徴

このタイプは「ポテンシャルの塊」。複数の研究（[参照](https://journals.lww.com/nsca-scj/fulltext/2020/10000/body_recomposition__can_trained_individuals_build.3.aspx)）により、多めにカロリーを取らなくても**筋肉を増やしながら脂肪を減らす（リコンプ）**ことが可能と示されている。

### なぜリコンプが可能か

1. **初心者効果**: 筋肉が刺激にめちゃくちゃ反応する
2. **脂肪エネルギーの蓄え**: 余剰エネルギーを内部から流用できる

### 推奨戦略

| 項目 | 推奨値 |
|------|--------|
| カロリー | メンテナンスカロリー維持、または **-5%** でもOK |
| タンパク質 | 体重1kgあたり **2.2g以上** |
| トレーニング | **週3回以上**、漸進的過負荷が必須 |
| その他 | 睡眠・ストレス管理を怠らない |

### メンテナンスカロリーの計算

- **簡易計算**: 体重(kg) × 33kcal（活動量ふつうの人）
- 例: 70kgの人 → 約2300kcal/日
- より正確には2週間ほど体重変化を見ながら調整

### 漸進的過負荷の例

| 方法 | 例 |
|------|-----|
| 重さを増やす | 100kg → 102.5kg |
| 回数を増やす | 8回 → 9回 |
| セット数を増やす | 3セット → 4セット |
| 休憩を短くする | 2分 → 90秒 |

### ⚠️ 注意点

リコンプは「ビギナーズラック」のようなもので長くは続かない。目安は**6ヶ月〜1年**で、その後はカロリー増量を視野に入れる必要がある。

## タイプ2：初心者＋痩せ型（余剰カロリー必須ゾーン）

### 該当条件

- 筋トレ歴が **1年未満**
- 男性: **体脂肪率12%未満**
- 女性: **体脂肪率20%未満**
- BMIが **18.5未満**（または標準だけど筋量が極端に少ない）

### 特徴

体脂肪という"予備タンク"がないため、筋肥大には外部からのエネルギー供給が不可欠。リコンプはほぼ起きない。「筋肉をつけるための"材料"も"燃料"も足りてない状態」であり、エネルギー不足だと身体は筋肉を削ってでも生存を優先する。

### 推奨戦略

| 項目 | 推奨値 |
|------|--------|
| カロリー | メンテナンス **+15〜20%**（+300〜500kcal/日） |
| タンパク質 | 体重1kgあたり **最低2.0g、理想は2.2g/kg以上** |
| トレーニング | **週3〜4回**で十分。ただし強度は高めに |
| 食事の質 | 好きなものの爆食いはNG。**高タンパク＋低脂質**の質重視 |

### 「食べられない問題」への対策

| 方法 | 解説 |
|------|------|
| リキッドカロリーを活用 | プロテイン＋オーツ、牛乳、豆乳、バナナなどをミキサーでまとめ飲み |
| 炭水化物は高密度系を選ぶ | 白米、餅、ドライフルーツ、バナナなど"少量で高カロリー"なもの |
| 脂質は「小容量・高カロリー」を選ぶ | MCTオイル、ナッツ、ナッツバターをスプーンでちょい足し |
| 間食をプロテインバーやグラノーラに | 小腹が空いたらプロテインチョコ、グラノーラ＋ヨーグルト |
| 分食スタイル | 1日3食にこだわらず、4〜6食の"こまめ食い"で胃の負担を減らす |
| トレ後30分以内に食事 | 筋トレ後は空腹感が出やすい"ゴールデンタイム" |
| 塩気を強めにして食欲を刺激 | 健康の範囲内で塩気を効かせる |
| 飲み物でカロリーを節約 | お茶・ブラックコーヒー中心にして"満腹ブロック"を防ぐ |

## タイプ3：中級者 × 体脂肪適正

### 該当条件

- 筋トレ歴: **1〜3年程度**
- トレーニングが **週3回以上**、コンスタントに続いている
- 体脂肪率: 男性 **12〜15%**、女性 **20〜25%**
- すでに**ある程度の筋肉量がある**（見た目に「引き締まった感」がある）

### 特徴

筋肥大ステージの"第2段階"。初心者のようにラクには筋肉が増えなくなる一方で、エネルギー不足によって成長が止まりやすい「中だるみステージ」に入りやすい。

### なぜちょい足しが必要か

- 体脂肪をエネルギー源として使いにくくなる
- 筋合成に必要な刺激が強くなり、回復にも多くのエネルギーが必要
- 代謝効率が良くなりすぎて、食べても太らないように"適応"してしまう

### 推奨戦略

| 項目 | 推奨値 |
|------|--------|
| カロリー | メンテナンス **+10%前後**（+200〜300kcal/日） |
| タンパク質 | 体重1kgあたり **2.2〜2.6g/kg** |
| トレーニング | 週4回をベースに、**徐々に負荷を上げていく** |
| 体重増加の目安 | **週あたり0.25〜0.5%**（70kgなら週+200〜300g） |

### ⚠️ 注意点

このステージから"脂肪がつきやすくなるリスク"が出てくるため、「数字より反応を見る」**オートレギュレーション戦略**が重要。

## タイプ4：上級者 × 絞れた体（リコンプ不可ゾーン）

### 該当条件

- 男性: **体脂肪率10〜12%**
- 女性: **体脂肪率18〜20%**
- トレーニング歴: **3年以上**
- 見た目はすでに「バキバキ、または引き締まっている」

### 特徴

**リコンプ（脂肪を減らしつつ筋肉を増やす）はほぼ不可能**。やるべきは**「増量期」と「減量期」の明確な切り分け**。

### 推奨戦略

| 項目 | 推奨値 |
|------|--------|
| カロリー | メンテナンス **+5〜10%** が無難 |
| タンパク質 | 体重1kgあたり **2.2〜2.5g/kg** |
| トレーニング | **強度とボリュームの両立**がカギ |
| その他 | **NEAT（非運動性熱産生）をできるだけ高める** |

### トレーニングのガイドライン

| 項目 | 内容 |
|------|------|
| 強度 | RM70〜85%の重量（8〜12回がギリの重さ） |
| ボリューム | 週あたり **10〜20セット／部位** を目安 |
| 頻度 | 各部位あたり週 **2回** が理想 |
| バリエーション | 時期に応じて、種目・可動域・収縮速度を変える |

### NEATを高める方法

| 方法 | 内容 |
|------|------|
| 歩数を増やす | 毎日8000〜10000歩を目標に |
| スタンディングデスク | 長時間の座位を減らす |
| インターバル活動 | 1時間ごとに1〜2分間の足踏み |
| 家事・掃除を積極的に | ちょっとした動作が蓄積する |
| 通話中は歩きながら | 動きながら話す習慣をつける |

### ⚠️ 注意点

NEATや代謝率が低下している可能性があるため「ちょっと増量しただけで太る」現象が起きやすい。ほんの少しずつのカロリー調整と緻密な反応の観察が大事。

## タイプ5：トレーニング再開組（マッスルメモリーあり）

### 該当条件

- 過去にしっかりトレーニングしていたが、**数ヶ月〜数年のブランク**がある人

### 特徴

"マッスルメモリー"が発動し、**驚くほどのスピードで筋肉が戻ってくるボーナス期間**に入りやすい。過去に筋トレをしていると、筋繊維に残っていた核の記憶がまだ生きているため、再刺激によって素早くタンパク合成が再起動する。

### 推奨戦略

| 項目 | 推奨値 |
|------|--------|
| カロリー | メンテナンス **±0%、または+5%程度** |
| タンパク質 | 体重1kgあたり **2.2g/kg以上** |
| トレーニング頻度 | **以前の70%くらいから再開**し、少しずつ戻す |

### ⚠️ 重要ポイント

- **スモールスタートが大事**: 最初から全開に戻すのはNG
- 関節・腱が追いつかないので「6〜7割ペース」から始める
- ブランク明けは「筋トレの精度」が落ちていることが多いので、**丁寧なフォームで神経系を再教育**する

## NEAT（非運動性熱産生）による"太りやすさ"の違い

### NEATとは

無意識の動き（貧乏ゆすり・立ち歩き・身振り手振りなど）によって生まれるカロリー消費のこと。

### 研究結果（[参照](https://pubmed.ncbi.nlm.nih.gov/9880251/)）

同じように1,000kcalの食事を追加で食べた場合：

- **NEATが高い人** → **700kcal**が消費される
- **NEATが低い人** → **100kcal**しか消費されない

→ **最大で6倍もの差**が出る

### NEATタイプチェックテスト

以下の質問に「YES」「NO」で回答：

| # | 質問 |
|---|------|
| ① | 座っているときも、つい足を動かしたり、貧乏ゆすりしてしまう |
| ② | 会話中はよく身振り手振りを使う（手がうるさいと言われたことがある） |
| ③ | エレベーターより階段を選ぶことが多い（2〜3階なら余裕で歩く） |
| ④ | 電話中や考えごとの最中に、無意識に歩き回るクセがある |
| ⑤ | スーパーやコンビニで、気づくと余計に店内を一周してしまう |
| ⑥ | 集中しているときでも、姿勢を頻繁に変えがち |
| ⑦ | 長時間座っていると体を伸ばしたり、ストレッチしがち |
| ⑧ | 混んでいる電車では、座るより立っている方がラクだと感じる |
| ⑨ | 机の上を頻繁に片付けたくなる（手を動かす口実を探しがち） |
| ⑩ | 暇になると「ちょっと掃除でもするか…」と身体を動かしたくなる |

### 判定ガイド

| YESの数 | タイプ | 特徴 |
|---------|--------|------|
| **8個以上** | 高NEATタイプ | エネルギーを勝手に燃やす"動くマシーン"。余分なカロリーをとっても自然に消費できる体質。太りにくい。 |
| **5〜7個** | 中NEATタイプ | やや活動的。意識次第で"太らずに食べる"が可能。日常の動きで差がつく。 |
| **4個以下** | 低NEATタイプ | "座ると根が生える"タイプ。余分なカロリーをとると脂肪が増えやすいので、意識的にNEATを増やす工夫が必要。 |

## まとめ：タイプ別の「食べ方ガイド」

| タイプ | 推奨カロリー | 推奨タンパク質 | 備考 |
|--------|-------------|----------------|------|
| 初心者＋体脂肪多 | メンテナンス〜-5% | 2.2g/kg以上 | リコンプ可能 |
| 初心者＋痩せ型 | +15〜20% | 2.2g/kg以上 | フルキングNG |
| 中級者 | +10〜15% | 2.2〜2.6g/kg | 最適サープラスゾーン |
| 上級者 | +5〜10% | 2.2〜2.5g/kg | 増量・減量を分ける |
| トレ再開組 | ±0%〜+5% | 2.2g/kg以上 | マッスルメモリー活用 |

## 重要な結論

> 「誰にでも効く増量法は存在しない」
>
> 増量は"あなたのタイプ・体脂肪・NEAT・トレーニング歴"によって、最適なやり方がまったく違う。**自分のタイプに沿って微調整する柔軟性**が必要。

## 事前準備

### 1. 全ての機能を有効にする

- **Logic Pro** → **設定** → **詳細** → 「全ての機能を有効にする」にチェック

### 2. サウンドライブラリのダウンロード

- **Logic Pro** → **サウンドライブラリ** から付属音源をダウンロード
- 有料音源を持っていない初心者は全てダウンロード推奨

## 1. ドラムの作成

### セッションプレイヤー（ドラマー）機能を使用

1. **トラック作成**: セッションプレイヤー → ドラマー → 作成
2. **BPM設定**: 今回は150に設定（ドラッグまたはダブルクリックで直接入力）
3. **リージョンの配置**: 5小節目から開始（前にイントロの余白を残す）

### ドラマー機能の設定

- **スタイル選択**: 音色と演奏スタイルを変更（例：ファンクロック）
- **複雑さと強さ**: スライダーで調整
- **詳細設定**:
  - スネア: 叩く部分を指定（センター等）
  - パーカッション: タンバリン、シェイカー、クラップを追加可能
  - ハイハット: 叩き方を指定（ハーフクローズ等）
- **フィル**: 量と複雑さを調整
- **フレーズガチャ**: 矢印マークをクリックして好みのフレーズを探す

### イントロのフィル作成

1. リージョンの左端をドラッグして1小節伸ばす
2. `Command + T` でカット
3. フィルの量と複雑さをMAXにしてガチャを回す

> **Tips**: リージョンを右クリック → 変換 → MIDIリージョンに変換で、後から微調整が可能

## 2. ピアノの作成

### セッションプレイヤー（キーボードプレイヤー）機能を使用

1. **トラック追加**: プラスボタン → セッションプレイヤー → キーボードプレイヤー

### コード進行の指定

1. グローバルトラックを表示（上部のボタンをクリック）
2. 右クリック → 「1つのグローバルトラックを選択」 → コードを選択
3. 再生バーを合わせて、プラスボタンでコードを追加

#### 今回のコード進行（Cメジャーキー）

| 小節 | コード |
|-----|-------|
| 5 | Fmaj7 |
| 6 | E7 |
| 7 | Am |
| 8 | C7 |
| 9〜11 | 繰り返し |
| 12 | Am |

### キーボードプレイヤーの設定

- **スタイル選択**: フリーリー等
- **複雑さ・強さ**: 調整可能
- **左手/右手のポジション**: 音域の高低を設定
- **ボイシング**: 細かな演奏スタイル
- **ムーブメント**: 音域の幅の動き
- **フィル**: 控えめに入れる（他の楽器の邪魔にならないように）

## 3. ベースの作成

### セッションプレイヤー（ベースプレイヤー）機能を使用

1. **トラック追加**: プラスボタン → セッションプレイヤー → ベースプレイヤー
2. コード進行は自動で反映される

### ベースプレイヤーの設定

- **スタイル**: ポップソングライター
- **複雑さ・強さ**: MAX
- **メロディ**: ルート以外の音をどれだけ入れるか
- **オクターブ**: オクターブ上の動きの量
- **フレーズ**: 音符の長さ

---

## 4. シンセ1の作成（マウス打ち込み）

### MIDIトラックの作成

1. **トラック追加**: プラスボタン → ソフトウェア音源 → 作成
2. **音色選択**: `Y` でライブラリを開く → シンセサイザー → クラシックス → アナログポリシンセ

### ピアノロールでの打ち込み

1. リージョンを作成（右クリック → MIDIリージョンを作成）
2. ダブルクリックでピアノロールを開く
3. `Command` を押しながらクリックで音符を入力（鉛筆ツール）
4. `Option + ドラッグ` でコピー

#### コード打ち込み例

| コード | 構成音 |
|-------|-------|
| Fmaj7 | ファ・ミ・ラ・ド |
| E7 | ミ・レ・ソ#・シ |
| Am | ラ・ミ・ラ |
| C7 | ド・ミ・ソ・ラ# |

### アルペジエーターの追加

1. トラックを選択 → **MIDI FX** → **アルペジエーター**
2. デフォルト設定でOK

### オクターブ調整

- `Command + A` で全選択
- `Option + Shift + ↑↓` でオクターブを移動

### 音量バランスの調整

- フェーダーを使って各トラックの音量を調整

## 5. シンセ2の作成（リアルタイム入力）

### トラックの複製

- MIDIトラックを選択した状態で `Command + D`

### 音色選択

- ライブラリ → シンセサイザー → プラックド → ノイジープラッドシンセ

### リアルタイム入力の方法

#### 通常の方法

1. スタート位置をループバーで指定
2. `R` で録音開始
3. MIDIキーボードで演奏

#### 裏技（推奨）

1. ループを少し前から始めて `Space` で再生しながら練習
2. 良い演奏ができたら `Shift + R`
3. Logicが裏でリアルタイム入力を記録しているので、それが取り込まれる

> **Tips**: ピアノが苦手な人はこの方法が効率的！

### クォンタイズ

1. `Command + A` で全選択
2. `Q` でグリッド線に揃える

### ベロシティ調整

- 全選択 → ベロシティスライダーで調整
- `Option + Shift + ドラッグ` で全て同じ数値に

---

## 6. リードの作成

### 音源の読み込み方法（ライブラリ以外）

1. 新規MIDIトラックを作成
2. **インストゥルメント** → **Alchemy** → **ステレオ**
3. 検索窓に「custom」と入力 → **Custom Triangle Lead 3** を選択

### メロディの打ち込みポイント

- **キーCメジャー**: 白鍵のみでメロディを作成可能
- **グレースノート**: ここぞというところに半音下の短い装飾音を入れると、しゃくりが入って良い感じになる

---

## 7. 仕上げ

### エンディングの作成

1. 曲の最後（12小節目）でピアノとリード以外をカット
2. `Shift + クリック` で複数選択 → `Command + T` でカット → 削除
3. ドラムリージョンを右クリック → 変換 → セッションプレイヤーに変換
   - 11小節目で自動フィルが入る

### リージョンに色をつける

- `Option + C` でカラーパレットを表示
- 楽器ごとに色を固定すると見やすい
  - 例: ドラム=赤、ピアノ=オレンジ、ベース=青、シンセ=紫、リード=ピンク

### マスタリングアシスタント

1. **マスタリング** ボタンをクリック
2. AIが自動で適切なエフェクトを提案
3. **ラウドネス** つまみを調整して、**LUFS** が **-9〜-10** になるように設定

> **重要**: この作業をしないと、YouTubeやSpotifyで配信した時に他の曲より音が小さく感じる

### さらに学ぶには

- **ミックス**: マスタリング前に行う重要な工程
- **ラウドネスノーマライゼーション**: 配信プラットフォームでの音量統一の仕組み

## まとめ

この動画で学べる主なポイント:

1. **セッションプレイヤー機能**: ドラム、ベース、ピアノをAIが自動演奏
2. **コード進行の指定**: グローバルトラックでコードを設定
3. **MIDI打ち込み**: マウスとリアルタイム入力の2つの方法
4. **基本ショートカット**: 作曲スピードを大幅に向上
5. **マスタリングアシスタント**: AIによる自動マスタリング提案

> DTMは難しそうに見えるが、セッションプレイヤー機能を活用すれば初心者でも本格的な曲を作ることができる。まずは同じ操作をしてみて、DTMの楽しさを体験することが大切。

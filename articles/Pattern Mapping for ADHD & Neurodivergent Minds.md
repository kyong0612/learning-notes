## 概要

**Unloop**は、行動パターンを視覚的にマッピングし、自己発見を促すツール。セラピーやコーチングではなく、**遊びと内省を通じたガイド付き自己発見**として位置づけられている。特に**ADHD**や**神経発達多様性（ニューロダイバーシティ）**を持つ人々に向けて設計されている。

---

## コア機能

### 1. ビジュアルパターンマッピング

- **9種類のノードタイプ**で思考、感情、行動、起源の記憶をキャプチャ
- リアルタイムで行動パターンが形成される様子を視覚化
- 非線形で、どこからでも開始可能

### 2. スタックポイント検出（Stuck Point Detection）

- AIがマッピング中に停止したポイントを検出
- **ブレイクスルーの質問**を生成（処方的なアドバイスではない）
- 自分で見えないものを見るための支援

### 3. X-Ray分析

- AIが隠れた関連性を明らかにする質問を生成
- アドバイスは提供せず、ユーザー自身の洞察発見を支援

### 4. 自己設計の実験（Self-Designed Experiments）

- ユーザーが自分でパターン介入を作成
- 処方的な解決策なし、完全なユーザー主導の変化プロセス

## ADHDの脳とパターンマッピングの親和性

### ADHDは「注意欠陥」ではなく「注意過剰」

> "Your ADHD brain is a pattern-recognition machine."
> （あなたのADHD脳はパターン認識マシンだ）

ADHDの脳は以下の点で優れている：

| 能力 | 説明 |
|------|------|
| **関連のないものを結びつける** | 関係ないはずのものの関連性を発見できる |
| **見えないものを発見する** | 誰も言及しない欠落を発見、当たり前を疑問視する |
| **時間旅行** | 過去・現在・未来を同時に認識し、因果関係を瞬時に把握 |
| **システムを感じる** | パターンを論理的にではなく感覚的に捉える |

### なぜ従来の自己啓発は失敗するのか

従来のアドバイス（プランナー、To-Doリスト、ポモドーロ・テクニック等）が失敗する理由：

> "These tools were designed by and for neurotypical brains. Brains that think in sequences."
> （これらのツールは定型発達の脳によって、定型発達の脳のために設計された。順序立てて考える脳のために。）

- **定型発達の脳**: 線形的・順序的に処理
- **ADHDの脳**: ウェブ状・爆発的・ジャズ的に処理

### ビジュアルマッピングが効果的な理由

| 特徴 | ADHDとの相性 |
|------|-------------|
| **非線形** | どこからでも開始可能、興味に従って進める |
| **全体が一度に見える** | ワーキングメモリに7つのステップを保持する必要がない |
| **動きがある** | パターンの流れ、ループ、スパイラルを視覚化 |
| **発見的** | マッピングしながらパターンを発見する |

## よくあるADHDパターン（誰も話題にしないもの）

### ⚡ Shame Spiral Speedrun（恥のスパイラル高速版）

```text
忘れる → 「私はバカだ」 → 恥 → 実行機能シャットダウン → もっと忘れる → 「なぜ私はこうなの？」
（所要時間：0.3秒）
```

### 🎢 Interest Oscillation（興味の振動）

```text
没頭 → 80%マスター → 飽きた → 放棄 → 辞めたことへの恥 → 新しいもの! → 「なぜ継続できないの？」
（永遠に繰り返し）
```

### ⏰ Waiting Mode Prison（待機モードの牢獄）

```text
午後3時に予定 → 何も始められない → 見失うかも → 待っている方がいい → 6時間の不安な無 → 疲弊 → それでも遅刻
```

### 💔 RSD Loop（拒絶感受性異常のループ）

```text
小さな拒絶（実際または想像） → 感情的爆発 → 過剰反応または引きこもり → 実際の拒絶を生む → 「ほら、やっぱり嫌われてる」 → さらに防御を固める
```

### 🧱 Executive Function Jenga（実行機能ジェンガ）

```text
管理できている → 追加が1つ → まだ管理 → もう1つ → まだ...大丈夫... → 小さなこと → 完全崩壊 → 「なぜ基本的な生活ができないの？」
```

## ADHDの脳と協力して働く方法

### The Messy Map Method（散らかったマップ法）

- カオスから始める
- ノードをあちこちに配置し、まだ接続しない
- 接続は自然に浮かび上がる

### The Hyperfocus Helper（過集中ヘルパー）

- 過集中が来たら、すべてをマッピング
- 細部まで、すべてのマイクロ接続を
- 後で簡略化できる

### The Squirrel Integration（リス統合法）

- 中断した思考もノードとして追加
- 気が散ったことはデータ
- 脱線は脱線ではなく、隠れた接続を示す脳のサイン

### The Energy Tracker（エネルギートラッカー）

- ノードにエネルギーレベルを追加
- この思考 = エネルギー消耗
- この行動 = エネルギーブースト
- ADHDはエネルギー管理の問題

### The Shame Translator（恥の翻訳者）

- 「私はバカだ」→「興味深い、恥がここに現れている」とデータ化
- 恥は、ADHDハードウェアで定型発達ソフトウェアを実行しようとするエラーメッセージ

## RSD（拒絶感受性異常）への対処

RSDはパターンマッピングを困難にする。すべてのパターンが個人的な失敗に感じられる。

### オーバーライド（上書き）の考え方

> "You're not mapping your character. You're mapping a system."
> （あなたは自分の性格をマッピングしているのではない。システムをマッピングしている。）

- パターンは**あなたが誰か**ではなく、**あなたが学んだこと**
- 学んだものは、学び直すことができる

## パターン変更におけるADHDの優位性

ADHDの脳はパターンを**見えるようになれば**、変更することに**優れている**：

| 特性 | パターン変更への効果 |
|------|---------------------|
| **新奇性追求** | パターンが見えると古くなり、新しいことを試したくなる |
| **高速反復** | 他の人が1つ試す間に17の実験ができる |
| **創造的解決策** | 非線形思考が想像もしない解決策を生む |
| **オール・オア・ナッシングの力** | 変えると決めたら、徹底的に変える |

## 実験アプローチの利点

ADHDの脳は実験に向いている：

| 特徴 | ADHDとの相性 |
|------|-------------|
| **短期間** | 実験には終了日がある。3日だけなら可能 |
| **新奇性が組み込まれている** | すべての実験は新しい |
| **規律ではなくデータ** | 「より良くなる」のではなく、データを収集 |
| **過集中との互換性** | 実験設計、実行、分析に過集中できる |

## エネルギー管理マップ

> "ADHD isn't actually about attention. It's about energy. Specifically, dopamine."
> （ADHDは実際には注意の問題ではない。エネルギー、特にドーパミンの問題だ。）

エネルギー注釈付きでパターンをマッピング：

- この思考 → エネルギー消耗
- この行動 → エネルギー中立
- この活動 → エネルギーブースト
- この交流 → エネルギー吸血鬼
- この成功 → エネルギー爆発

---

## 時間盲のブレイクスルー

ADHDの時間感覚：

- **今**
- **今ではない**

これだけ。

マッピング時にタイムスタンプを追加することで発見できること：

- 永遠に感じることは5分で終わる
- 一瞬に感じることは3時間かかる
- 毎日起こると思っているパターンは週1回
- 稀だと思っているパターンは常に起こっている

## 重要な結論

1. **ADHDは欠陥ではなく、異なるオペレーティングシステム**
   - WindowsのバグではなくLinux - 違うだけで間違っていない

2. **パターンは性格の欠陥ではなく、適応**
   - 異なる脳のために構築された世界で生きるための適応

3. **失敗した試みはデータ**
   - 自己改善に失敗したのではなく、何がうまくいかないかのデータを収集していた

4. **ビジュアルマッピングは脳のネイティブ言語**
   - ADHDの脳は自然にパターン、接続、視覚的関係で考える
   - 最初からこの言語を話していた。今はそれを描くキャンバスがある

## 補足：セラピーとの違い

Unloopは明確にセラピーや精神健康治療**ではない**と位置づけている：

- 自己ガイド型で洞察主導
- 行動を視覚的に探索するのを助けるが、診断・処方・セラピーの代替はしない
- パターンを明確に見るのを助ける「内省的なコンパニオン」として機能

## バイオハック６．サーチュイン活性化食材を取り入れる

アンチエイジングについて、ここ数年でよく言われるのが「サーチュイン遺伝子」です。サーチュインとは、ひとことで言えば「長寿遺伝子」として知られる酵素群のこと。NAD+（ナイアシンの一種）を材料に使いつつ、細胞のダメージ修復や代謝の最適化を担っている酵素で、ざっくりいうと「細胞を若返らせるスイッチ」みたいな存在です。

体内には全部で7種類のサーチュイン（SIRT1〜SIRT7）が存在しており、それぞれが以下のような多彩な役割を担っています：

- DNA修復
- ミトコンドリアの再生
- 炎症の抑制
- アポトーシス（細胞の自滅）制御
- サーカディアンリズム（体内時計）の調整
- エネルギー代謝の最適化

なかでも有名なのはSIRT1で、これは「カロリー制限をすると寿命が延びる現象」と密接な関係があるとされています。長寿遺伝子の本命とも言われる存在です。

このスーパー酵素たちを活性化するには、カロリー制限や運動（特に有酸素＋HIIT）が基本。特に有酸素運動と高強度インターバルトレーニング（HIIT）の組み合わせは、NAD+合成酵素（NAMPTなど）が増え、サーチュインを活性化してくれやすいので、これは日々心がけておきたいですね。

これらの基本をふまえた上で、サーチュインをオンにできる食べ物を見ていきます。研究の世界では「STACs（Sirtuin-Activating Compounds）＝サーチュイン活性化化合物」と呼ばれる物質のことで、いまのところ以下の食材が有望視されています([R](https://www.sciencedirect.com/topics/medicine-and-dentistry/sirtuin-activating-compound))。

| STACs（活性化化合物） | 主な含有食材・食品（Sirtfoods） |
|----------------------|--------------------------------|
| アピゲニン | パセリ、ジャーマンカモミール、ローマンカモミール、スイートオレンジ、ウィートグラス |
| カプサイシン | トウガラシ類（カプシカム） |
| チコリ酸 | チコリ |
| クルクミン | ターメリック（ウコン） |
| エピガロカテキンガレート | 緑茶、ウーロン茶 |
| ヒドロキシチロソール | オリーブオイル |
| ケルセチン | りんご、紅茶、ソバ、ケーパー、さくらんぼ、ぶどう、ケール、ラヴァッジ、玉ねぎ、ラズベリー、いちご |
| レスベラトロール | ビルベリー、ブルーベリー、カカオ、クランベリー、ぶどう、ワイン |
| ニコチンアミドリボシド | ビール酵母、ホエイプロテイン |

いずれも「パレオな男」では定番の食品ですが、これらはどれも「カロリー制限ミメティック」、つまり断食やカロリー制限と似たような効果を細胞に与えると言われており、その意味でもかなりアツい注目を集めています。

## フィセチンが熱い件

近年特に話題になっている「フィセチン」というポリフェノールを押さえておきましょう。フィセチンは、イチゴ、リンゴ、柿、玉ねぎなどに含まれる成分で、

- 老化細胞（＝セネセント細胞）を除去する"セノリティクス"として作用する([R](https://pubmed.ncbi.nlm.nih.gov/30279143/))
- その結果として、炎症・細胞機能の低下・幹細胞の機能低下を抑える

といった、"体の大掃除役"のような働きをしてくれることがわかってきました。最近の老化研究の文脈では、「老化細胞（＝セネッセント細胞）」をどう取り除くか？がかなりホットな話題になっています。今のところ、フィセチンについてはヒトの実験が少ないですが、動物実験([R](https://www.sciencedirect.com/science/article/pii/S2352396418303736))では「フィセチンを投与されたマウスの平均寿命が10%〜36%延びた！」という報告が出ています。

ちなみに、老化細胞について簡単に説明しましょう。年を取るとともに、私たちの細胞の一部は「もう分裂しません！」と言い出し、かといって体内からいなくなるわけでもなく、体の中に居座るようになります。これが「老化細胞」で、

- 炎症物質をまき散らす
- 周囲の健康な細胞に悪影響を及ぼす
- ガンや動脈硬化、認知症のリスクを高める

…と、かなり迷惑な行動に出てくるんです。うーん、老害。

そんな老化細胞を"選択的に排除する"物質は「セノリティック」と呼ばれ、現在も数多くの候補が研究中ですが、その中でも「効果が高くて、副作用が少ない」とされているのが、イチゴやキュウリなどに含まれるポリフェノールの「フィセチン」です。

## セノリティック・ファスティングのやり方（サプリ使用）

個人的には、断続的にフィセチンのサプリを摂る「セノリティック・ファスティング」的な使い方がおすすめです。

- **使用頻度**：月に1回 or 2回
- **期間**：1回のセッションで2日連続
- **フィセチンの量**：1日あたり20mg/kg〜50mg/kg（体重1kgあたり）→ 体重60kgなら 1,200〜3,000mg程度。フィセチンは脂溶性なので、「食事と一緒に摂る」か、「MCTオイルや魚油などと一緒に摂る」のが吸収を高めるコツ。

たとえば以下のようなスケジュールになります：

- 土曜に1,500〜3,000mgのフィセチン
- 日曜に1,500〜3,000mgのフィセチン

なぜこういうやり方をするのかというと、フィセチンは天然成分とはいえ、高用量の長期連用は推奨されていないからです。老化細胞は完全に"老害"というわけでもなく、一部の老化細胞は、傷ついた細胞の修復や再生に関与することもあります。そのため、セノリティックは毎日摂るようなものではなく、あくまで「掃除日」的に取り入れるのがベターです。

「セノリティック・ファスティング」をやった翌日に「若返った！」という実感はまずないのでご注意ください。人によっては、2〜3ヶ月スパンで見た場合に、「疲れにくくなった！」「肌の調子が良くなった！」「回復力が上がった！」みたいな実感を得るケースもあるようです。

フィセチンを使ってみたい方は、低用量の商品なら「[Doctor's Best, フィセチン](https://jp.iherb.com/pr/doctor-s-best-fisetin-100-mg-30-veggie-caps/43592?rcode=IRO489)」、がっつり高用量でいきたいなら「[Humanx, フィセチン](https://jp.iherb.com/pr/humanx-fisetin-500-mg-30-capsules/132363?rcode=IRO489)」を使ってみてください。

## セノリティック・ファスティング＋ケルセチン

さらにセノリティックにこだわりたいなら、フィセチンとケルセチンを併用するのも非常に良い方法です。両者とも老化細胞を除去するセノリティック作用をもつポリフェノールの一種ですが、作用機序がやや異なるため、併用すると互いを補完し合い、効果を高め合うとされています。

特に注目すべきは：

- **フィセチン**：単体でも老化細胞をそこそこ除去できる可能性が高い
- **ケルセチン**：フィセチンの吸収をサポートし、異なる経路で老化細胞を叩く可能性がある（特に脂肪組織など）

ケルセチンによってフィセチンの働きがブーストされる可能性がかなりあります。ケルセチンはブロッコリーや赤タマネギ、リンゴ、ぶどう、緑茶などに含まれる抗アレルギー＆抗炎症物質です。

海外の研究者やバイオハッカーが実践している「2日間集中コース」は以下のように行います：

| 項目 | 内容 |
|------|------|
| 実施日数 | 2日間（連続） |
| 食事 | 朝～夕にかけて軽めの低糖質食 or プチ断食（オートファジー促進） |
| サプリ投与例 | **フィセチン:** 1,000〜1,500mg /日（体重60〜70kgの人の場合）<br>**ケルセチン:** 500〜1,000mg /日<br>**NAC:** 500〜600mg（ケルセチンの活性サポート）<br>**MCTオイル or フィッシュオイル:** 吸収性アップ用に少量 |
| 飲み方のコツ | 朝食前か昼前にまとめて摂取（吸収効率◎）<br>必ず脂質（オイルや食事）と一緒に摂取<br>水分をしっかりとる<br>翌日はしっかり睡眠・休息を確保 |

※NAC＝N-アセチルシステイン：抗酸化と肝サポートのためによく使われ、特にケルセチンとのシナジーが報告されています。ただし、その効果はよくわからないので、あえて摂取しなくても良いでしょう。

実際の摂り方のポイント：

- 朝食前か昼前に一括で摂るのが一般的（吸収効率が高い時間帯）
- 必ず脂質を含む食事やオイルと一緒に（脂溶性だから吸収UP）
- 水分は多めにとる（代謝の負担軽減＋老化細胞の排出サポート）
- セノリティックの翌日はしっかり睡眠＆休息（排出と修復が進む時間）

このスタックが効くかどうかはあくまで"理論上"のものなので、まだヒトを対象にした試験はないことをご承知ください。

## まとめ

今回はサーチュインの活性化に役立つ食材を見てみました。ポイントは「調味料的にちょっとずつ＆習慣的に」食材を取り入れることです。例えば：

- 朝のスムージーにブルーベリーパウダー or 冷凍野生ベリーを1日1回
- 週2〜3回ほど無糖ダークカカオ＋MCTオイル
- 緑茶や抹茶を1日2～3杯を目安に飲む（カフェイン過敏の方は注意）
- クルクミンやフィセチンのサプリを週数回ローテーションで

抗酸化成分を摂りすぎると逆に細胞のストレス耐性（ホルミシス）が落ちる可能性があるので、薬味的に使うのがポイントです。ここらへんが面倒なら、まずは冷凍ベリーと緑茶から試すのも良いでしょう。

長らく続いてきた「脳の最適化」シリーズはこれで終了です。情報量が多いので、自分がやってみたいと思えるものから手を付けていただければ幸いです。

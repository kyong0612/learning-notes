## 「ネガティブな脳」を修正する最新テクニック「CBM-M」をやってみよう！#2「実践編」


『「CBM-M」をやってみよう！』の続きです！([#1](https://ch.nicovideo.jp/paleo/blomaga/ar2229254))

簡単におさらいしておくと、「CBM-M」ってのは、ポジティブな記憶を意図的に思い出すトレーニングをすることで、「ネガティブな出来事ばかり思い出してしまう脳のクセ」を少しずつ修正していく脳トレの一種であります。こいつを使うと、「嫌な出来事ばかり思い出して気分がへこむ……」みたいな経験を克服できると考えられるんですな。

不安が強い人や落ち込みやすい人ってのは、どうしても「失敗」「恥」「後悔」みたいな出来事を優先して思い出しやすいわけですが、このCBM-Mでは「安心」「感謝」「達成」などのポジティブワードを手がかりにして、前向きな自分の体験を繰り返し呼び出していくんですよ。こうした作業を続けることで、脳の「記憶検索のアルゴリズム」が徐々に調整され、ネガティブな出来事に引きずられにくくなり、結果としてストレス耐性が高まる……みたいな考え方ですな。

## ストレスに強くなる記憶トレーニング：「CBM-Mセルフ実践プロトコル」

では、ここまでの話を押さえたところで、いよいよCBM-Mを実践してみましょう。ここから紹介するのは、論文のプロトコルをもとにしつつ、家庭でも一人で実践できるように少しアレンジした「セルフ版CBM-Mトレーニング」であります。心理実験そのままの厳密な手順じゃないんだけど、基本的なメカニズムは同じなので「ネガティブな記憶の検索クセ」を調整するトレーニングとしては十分役に立つはず。紙とペン、あるいはスマホのメモがあればすぐに始められるので、気軽に試してみてくださいませ。

このCBM-Mセルフプロトコルは、1ヶ月（週2回 × 4週間）で不安の改善を目指す内容になってまして、1回あたりのセッションで必要な時間は約15〜20分ぐらい。ポジティブな自伝的記憶（自分に関する前向きな体験）を強化し、ストレス反応・コルチゾール・ネガティブ記憶バイアスを下げるのがゴールになります。具体的には、以下のステップで取り組んでみてください。

### ステップ 0. 下準備

まずポジティブワードのリストを用意しましょう。研究でよく使われるのは、

- 自己評価に関係する言葉
- 感情や安心感に関係する言葉
- 社会的つながりに関係する言葉

の3タイプでして、「自分の体験と結びつきやすい言葉」のほうが上手く働いてくれます。CBM研究でよく使われるタイプの単語をまとめてみると、以下のようになります。

## 基本ポジティブワード（30語）

- 安心
- 達成
- 成功
- 感謝
- 誇り
- 満足
- 幸福
- 希望
- 楽しい

この中から、1回のセッションで10語ぐらいランダムに選んで使ってみてくださいませ。

また、CBM-Mの効果ってのは、 自分の体験とリンクしやすい言葉を使うほどアップするので、自分でワードリストを作ってみるのもかなり有効であります。その際は、次の問いに答えて、思い浮かぶ言葉を書き出してみてください。

## CBM-M用の質問リスト

- 最近うまくいったことは？
- 人に感謝されたことは？
- 自分の強みは？
- ほっとした瞬間は？
- 楽しかった時間は？
- 自分を少し好きになれた瞬間は？

でもって、これらの質問の答えから単語を抜き出してみてください。たとえば、

- 「プレゼンがうまくいった」
- → 達成
- → 成功
- → 自信
- 「友人に相談に乗ってもらった」
- → 信頼
- → 友情
- → 安心

みたいになります。こうして自分用ポジティブワードを10〜20個追加すると記憶が呼び出しやすくなりますんで、余裕がある方はこちらもお試しくださいませ。

### ステップ 1. マインドセット（1分）

まず最初に行うのが「マインドセット」から。ここでやることはシンプルで、これから行うトレーニングは **単なるポジティブ思考の練習ではなく、「記憶の検索パターン」を調整するトレーニングなのだ！** と意識することです。

人間の脳ってのは、過去の出来事をそのまま保存しているわけではなく、 その都度「検索」して思い出す仕組みになってまして、不安やストレスが強いときほど、 この検索システムはネガティブな記憶を優先して引き出すようになるものなんですよ。この意味で、CBM-Mってのは、その検索アルゴリズムを少しずつ書き換えていく作業だと考えてください。

セッションを始める前には、できるだけ外部の刺激を減らしておくのがポイント。 スマートフォンの通知をオフにし、静かな場所に座り、背筋を軽く伸ばして呼吸を整えましょう。このとき、30秒ほどゆっくり呼吸するだけでも集中力が上がりますんでお試しを。

また、ここで「良い記憶をうまく思い出さねば！」と力む必要はありませんで、このトレーニングは記憶の内容を評価するものではなく、あくまで **ポジティブな体験を検索する習慣を脳に学習させること** が目的なので、そこは肝に銘じておきましょう。

最後に、「今日は自分の脳の検索エンジンを少しアップデートする時間だ！」と軽く意識してから セッションに入ってくださいませ。この短い準備を入れることで、 注意の集中が高まり、ポジティブな記憶の想起がスムーズになりますんで。「こんな作業必要か？」とか思うかもですが、トレーニング全体の質を左右する重要なステップなんで、ぜひお忘れなく。

### ステップ 2. 単語提示（約5分）

このステップでは、ステップ０で用意したポジティブな単語をきっかけにして、 **自分に関する前向きな記憶を意図的に呼び出す** 練習を行います。ここがCBM-Mの中心でして、要するに「脳がネガティブな出来事ばかり優先検索するクセ」を、 少しずつ修正していくパートだとお考えください。

前回も説明したとおり、不安が強い人や落ち込みやすい人ってのは、「失敗」「恥」「後悔」みたいな記憶が自動で出やすいわけですが、 このステップでは逆に、「安心」「感謝」「達成感」「有能」といった単語を足がかりにして、 ポジティブな記憶の検索回路を鍛えていくことになります。

このステップを実践する際は、まずはポジティブな単語をランダムで選んで、その言葉をじっと見てみましょう。ワードをランダムで確認する方法はいくつかありまして、以下から好きなものをお選びください。

## 紙カード法（定番）

最も定番なのがこれ。アナログですが、実はかなり使いやすい方法です。

### やり方

1. ポジティブワードを20〜30個書く
2. 単語をカードや紙に1つずつ書く
3. 箱に入れる
4. シャッフルして1枚ずつ引く

心理実験でも、カードシャッフル方式は普通に使われますんで、とりあえずこれを試すのがおすすめっすね。

## Googleスプレッドシート法

こちらも簡単なのにかなり便利な方法で、まずはGoogleスプレッドシートの1列に、ステップ０で作ったポジティブワードを書きます。たとえば、

> ```
> A1 安心
> A2 成功
> A3 感謝
> A4 成長
> A5 誇り
> ```

みたいな感じですね。その上で、別のセルに、

> ```
> =SORT(A1:A30,RANDARRAY(30))
> ```

と入力してくださいませ。これだけで単語が毎回ランダム並び替えされますんで、その上から10個を使えばOKです。

## ランダム単語アプリ

スマホ用のランダム単語アプリもいくつか出てますんで、こちらを使うのもあり。たとえば、

- Tiny Decisions
- Randomizer
- Random Name Picker

あたりが有名なので、検索して気にいったものをお使いください。どれも使い方は簡単で、ポジティブワードを登録してからスイッチをタップするだけなんで。

## ChatGPTを使う

これもかなり便利で、たとえば 「ポジティブワードをランダムで10個出してください」 と入力するだけで、毎回違う単語が出てきたりします。これでもまったく問題ないっすね。

さて、以上の方法のどれかを使って任意のポジティブワードを見たら、 **その言葉に結びつく **具体的な自分の体験** を思い出してみましょう** 。たとえば「有能」という単語を見たなら、 「以前、難しい仕事をなんとか終わらせた日」や 「誰かに頼られて役に立てた瞬間」などを思い描きます。 「感謝」なら、 「誰かに親切にしてもらってホッとした場面」や 「自分が感謝されたときの記憶」でも構いません。

ここで重要なのは、「ポジティブな意味を頭で考える」だけで終わらせず、 **実際にあった一場面をできるだけ鮮明に思い出す** こと。思い出す際には、なるべく以下の要素を入れてみてください。

- いつの出来事だったか
- どこで起きたか
- 誰がいたか
- 自分が何をしたか
- そのとき何を感じたか

たとえば「達成感」という単語を見たなら、 「去年の夏、夜中まで準備したプレゼンがうまくいって、終わったあとに肩の力が抜けた」 みたいな感じで、情景つきで思い出すのが理想です。 ぼんやり「まあ、昔いいこともあったかな……」ぐらいの記憶だと効果が弱いので、 できるだけ一つの場面をありありと思い出すほうが、記憶の再学習が進みやすくなりますんで。

また、このステップでは「すごい成功体験」だけを探す必要はありません。 むしろ、

- 朝ちゃんと起きられた
- 人にやさしくできた
- 面倒な作業を片づけた
- 少し気分が軽くなった時間があった

みたいな、小さなポジティブ記憶で十分であります。 CBM-Mの目的は、自慢できるエピソードを集めることではなく、 **自分の人生の中から前向きな記憶を検索する習慣をつくること** にありますんで、「こんなの大したことないしな……」と切り捨てないのがめっちゃ大事。 脳は反復された検索パターンに影響されるので、小さくても具体的な記憶を何度も呼び出すほうが効くんですよ。

1つの単語に使う時間は、だいたい20〜30秒くらいでOK。長く考え込みすぎると疲れちゃうので、 「まずはひとつ浮かんだ場面を採用する」ぐらいの軽さで進めましょう。必要なら、思い出した内容を1〜2行メモしておくのもおすすめです。メモを書くことで記憶の輪郭がはっきりして、後の「ワードリコール」でも思い出しやすくなりますんで。

もし単語を見ても何も浮かばない場合は、無理にひねり出さなくても大丈夫です。そのときは、「完全に単語と一致する体験」ではなく、 「少し近い感覚があった場面」まで範囲を広げてください。 たとえば「誇り」でピンと来なければ、 「ちょっとだけ自分を悪くないと思えた瞬間」でも十分です。

### ステップ 3. ワードリコール（約3分）

続いて行うのが「ワードリコール（単語の想起）」です。ここでは、先ほど見たポジティブワードを、 できるだけ多く思い出して書き出してください。紙やメモアプリを使い、 思い出せた単語をそのまま書き出していきましょう。もし10個すべて思い出せなくても問題はありません。 このトレーニングはテストではないので、「どれだけ思い出せたか」よりも「思い出そうとする行為」が大切です。重要なのは、思い出す順番や正確さにこだわることではなく、 **自分の記憶の中から言葉を検索するプロセスそのもの** にありますんで。

このステップの狙いは、ポジティブな記憶の検索回路を繰り返し使うことで、 **脳の検索アルゴリズムをポジティブ方向に調整する** ことです。 普段、ストレスや不安が強いときには、脳はネガティブな情報を優先して検索しちゃうものなんですが、このようにポジティブな単語を手がかりにして思い出す練習を続けることで、 ポジティブな記憶がアクセスしやすい状態へと徐々に変わっていきます。

実践するときのコツとしては、思い出せない単語があっても焦らないことです。思い出そうとして数秒考えてみて、それでもなにも浮かばなければ次の単語に進んで構いません。また、思い出した単語に関連する体験が自然に浮かんできた場合は、その感覚を軽く味わってみてください。このプロセスを数分間繰り返すことで、ポジティブな記憶の検索回路が少しずつ強化されていきます。

### ステップ 4. 連想拡張（約2分）

「連想拡張」ってのは、ポジティブな単語から思考を広げていくことで、脳の「ポジティブ連想ネットワーク」を強化するためのワークです。CBM-Mの基本トレーニングでは、単語からポジティブな記憶を思い出すトレーニングが最も重要なパートですが、この連想拡張はもう一歩進めて、ひとつのポジティブ単語から複数の前向きな連想を引き出すことを目的にしております。脳は、ある言葉をきっかけに関連する記憶やイメージを次々に呼び出す性質を持ってますんで、この仕組みを意図的に利用するわけですな。

やり方はシンプルで、

1. まず、紙カード箱からポジティブワードのカードを1枚引く（もちろん、GoogleスプレッドシートでもアプリでもOK）。
2. その単語を見たら、そこから連想されるポジティブな言葉やイメージを3つほど書き出す。たとえば「感謝」というカードを引いたなら、「家族」「友人」「サポート」のように思いついた単語を書き出すか、「助けてもらった出来事」「誰かにお礼を言った場面」といった具体的な体験のいずれかを書き出す。

って感じになります。重要なのは、頭に浮かんだ前向きな連想をできるだけ素早く書き出すことです、2〜3秒で思いついたものを書くようにしてください。

この作業を続けていると、ポジティブな言葉が単独で存在するのではなく、さまざまな経験や人間関係、感情と結びついていくのが実感できるはず。心理学ではこれを「意味ネットワーク」と呼びまして、ポジティブな連想が増えるほど、このネットワークが広がり、前向きな記憶や感情を呼び出しやすくなるんですな。ネガティブな出来事ばかり思い出してしまう人は、このネットワークが狭くなっていることが多いんで、このトレーニングで偏りを調整してみるといい感じです。

実践の目安としては、カードを1枚引くごとに3つの連想を書き出し、それを5〜10回ほど繰り返すとよいでしょう。シンプルなワークですが、続けていくうちにポジティブな記憶を検索するスピードが上がり、日常のストレスに対する心理的な耐性も高まりやすくなるはずであります。

### ステップ 5. 気分チェック（1分）

最後に、以下の2つを1〜9点で自己評価しましょう。

- いまの気分はどれくらい心地いいか？（1＝最悪、9＝最高）
- 今日のセッションは自分にとってどれくらい意味があったか？（1＝意味なし、9＝非常に意味あり）

これにてCBM-Mの全ステップは終了です。いちおう「ステップ 4. 連想拡張」はオプションのワークなんで、めんどうなら飛ばしても構いませんけども、ガッツリと効果を得たいならこれもやっといてください。

進め方のガイドラインとしては、

- **1週目（1〜2回目）** ：過去の「ちょっとした成功体験」から始める
- **2週目（3〜4回目）** ：「人とのつながり」に関するポジティブ記憶を扱う
- **3週目（5〜6回目）** ：「自分の強み」や「感謝」に関連する体験を掘る
- **4週目（7〜8回目）** ：「困難を乗り越えた体験」や「成長実感」にフォーカスする

みたいにやっていくといいでしょう。その上で、全8回のセッションが終わったら、以下の用量で効果測定（セルフチェック）を行ってください。

1. 最近、ネガティブな出来事を思い出す頻度（1〜10）
2. ポジティブな出来事を思い出せる速さ（1〜10）
3. 日中の疲労感・だるさの度合い（1〜10）

これら3つのポイントを、1回目の前と8回目の後で評価して比較すればOKであります。おそらく、多くの人はネガティブ記憶のバイアスが減ったり、なんとなく疲労感が軽減したのを実感できるんじゃないでしょうか。

最後に注意点を述べておくと、繰り返しになりますが、前回解説したCBM-M研究では、「うつ傾向が強い人」がこの作業を行った場合は、逆にポジティブ記憶が減る場合もあったとされております。なので、うつ状態に該当しそうな方は、以下のポイントを守るのが無難でしょう。

- 無理に「楽しかったこと」を思い出そうとしない
- 「過去の自分にねぎらいをかけるような記憶」から始める
- 可能なら専門家のサポートを受けながら実施する

## まとめ

ってことで、CBM-Mの話はこれで終了です。人間の脳は「過去に何を思い出すか」で未来の感情が変わるようにできてますんで、このトレーニングによって感情を押し殺すのはなく、「記憶の取捨選択スキル」を鍛えれば、あなたのレジリエンスが高まる可能性は高いでしょう。たった1ヶ月でも脳の配線は変わりますんで、ぜひお試しください。

ちなみに、そこまで不安が強くないけれどもCBM-Mのエッセンスだけ取り入れてみたいという方は、

- 夜に反省会をしない
- 1日の終わりに「今日は何がうまくいったか」を具体的エピソード付きで1つ思い出す

って作業を実践してみるだけでも意味があるはず。これでも脳の「検索設定」を少しずつ書き換えられるはずであります。お試しあれー。

## 概要

**Code Wiki** は、Googleが提供するAI駆動のコードドキュメントハブ。リポジトリのソースコードを専門のAIエージェントが分析し、自然言語での要約、アーキテクチャ図、その他の重要な洞察を自動生成する。開発者が新しいコードベースを素早く理解し、オンボーディングを加速させ、より早く貢献できるようにすることを目的としている。

## 主要機能

### 1. セクション別コード理解

- 関心のあるコード部分にフォーカスして、その動作を深掘り可能
- 複雑なシステムをセクションごとに分解して理解

### 2. 自動生成ドキュメント

- AIエージェントがコードから自動的にリッチでインタラクティブなナレッジベースを生成・維持
- 手動でのドキュメント作成が不要

### 3. 常に最新の状態

- プルリクエストがマージされるたびに、関連ドキュメントが自動更新
- 古いドキュメントの問題を解消

### 4. コードへのリンク

- アーキテクチャの概要から特定のサービスへ、関数の説明からリポジトリ内の定義へ即座にジャンプ可能
- シームレスなナビゲーション体験

### 5. ダイアグラム生成

- コードベース内の関係性、階層、依存関係を視覚的に表現
- インタラクティブなマップでズーム・パン・探索が可能
- 複雑なシステムを直感的なビジュアルで理解

### 6. コードベースとの対話（Chat機能）

- 自然言語でアーキテクチャに関する質問が可能
- 関数定義の検索、複雑なロジックの理解を会話形式でサポート
- 24時間365日利用可能な「エンジニア」のような存在

## Geminiとの関係

Code WikiはGoogleのGeminiモデル上に構築されている。専門のAIエージェントがGeminiを使用してソースコードを分析し、Wiki内のすべてのコンテンツを生成する。「Ask about this repo」機能では、Wikiのコンテンツを主要なコンテキストソースとして使用するGemini搭載のチャット体験を提供。

## 利用方法

### 無料利用

- 公開オープンソースリポジトリのCode Wikiへのアクセスは完全無料

### Wikiの検索方法

1. **検索**: codewiki.googleホームページの検索バーでリポジトリを検索
2. **直接URL**: リポジトリURLの先頭に `codewiki.google/` を追加
   - 例: `codewiki.google/github.com/owner/repo`

## サポートプラットフォーム

- **現在**: GitHub のみ
- **将来予定**: GitLab、Bitbucket

## プライベートリポジトリ対応（Coming Soon）

- プライベートコード用に、Gemini CLI向けのCode Wiki拡張機能をリリース予定
- 自身の環境内で安全にWikiを生成・ホスト可能
- 通知登録が可能

## 更新頻度

- Wikiは定期的に更新され、ソースリポジトリの変更を反映
- リフレッシュレートは人気度、使用状況、その他の要因に基づいて動的に変化
- 最終生成日はページ左下で確認可能

## 制限事項

- ドキュメントは自動生成のため、直接編集やプルリクエストは受け付けていない
- 現時点ではGitHubリポジトリのみサポート
- プライベートリポジトリは未対応（拡張機能で対応予定）

## サポート・ポリシー

### リポジトリのオプトアウト

リポジトリオーナーがCode Wikiの削除を希望する場合は、<support-codewiki@google.com> にメールで連絡可能。

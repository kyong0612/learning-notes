## 要約

### 概要

Claudeがユーザーのヘルスデータと安全に連携できる新機能を発表した。4つの新しい統合がベータ版として利用可能になった。

### 4つの新しい統合

1. **Apple Health** (iOS) - iOSデバイスの健康データと連携
2. **Health Connect** (Android) - Androidデバイスの健康データと連携
3. **HealthEx** - ヘルスケアプラットフォームとの連携
4. **Function Health** - Function Healthサービスとの連携

### プライバシー設計

- **明示的なオプトイン**が必要 - ユーザーが自ら選択して連携を有効化
- **トレーニングへの不使用** - ヘルス情報はAIのトレーニングには一切使用されない

### 利用可能な機能

連携後、Claudeは以下のことが可能になる：

- 医療履歴の要約
- 検査結果をわかりやすい言葉で説明
- フィットネス指標のパターン検出
- その他のヘルスデータ分析

### ベータ版の利用条件

- **対象ユーザー**: ProおよびMaxユーザー
- **対象地域**: 米国のみ
- **利用開始方法**:
  - iOS/AndroidのClaudeアプリから開始
  - HealthExとFunction Healthへの接続: https://claude.com/connectors

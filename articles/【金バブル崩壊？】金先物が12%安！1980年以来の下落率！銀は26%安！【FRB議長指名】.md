## 概要

高須幹弥氏（高須クリニック）が、2026年1月30-31日に発生した金価格の急落について自身の見解を解説した動画。金先物が12%下落（1980年以来の下落率）、銀は26%下落という状況に対し、下落の原因と今後の見通しを分析している。

## 金価格急落の原因

### FRB議長指名の影響

- **トランプ大統領がケビン・ウォーシュ氏を次期FRB議長に指名**
- ウォーシュ氏は**金融緩和に消極的なタカ派**
- 市場予想：トランプ大統領はイエスマン（利下げ要求に従う人物）を指名すると見込んでいた
- 現実：タカ派が指名されたことで予想が覆された

### 市場への連鎖的影響

1. **利下げが進まない見通し** → 市場予想よりドル高へ
2. **ドル高** → ドルインデックス上昇 → **金価格下落**
3. **ドル高** → 円安 → 日経平均先物上昇

### トランプ大統領の懸念

- 株安・国債安・ドル安の「トリプル安」を絶対に避けたい
- 長期金利上昇も懸念材料（FRBは短期金利を決定するが、長期金利はマーケットが決定）
- マーケット動向を見て発言を即座に撤回する柔軟性がある

## 金価格上昇の背景要因

### 直近の金価格動向

- **1月29日時点：1g = 3万円突破**
- **4ヶ月で1万円上昇**という急騰
- 12%下落しても、チャート上は2週間前の水準に戻っただけ

### 構造的な上昇要因

| 要因 | 詳細 |
|------|------|
| **ドル離れ** | ロシアのウクライナ侵攻時にドル資産が凍結 → 米国と敵対する国がドルから金へシフト |
| **中央銀行の金買い増し** | 各国が外貨準備としてドルを売って金を購入 |
| **地政学リスク** | ベネズエラ情勢、戦争リスクの高まり |
| **法定通貨供給量増加** | 日本・米国ともにインフレ傾向 |
| **関税リスク** | トランプ大統領のヨーロッパ8カ国への10%追加関税宣言（後に撤回） |
| **ドル基軸通貨の地位揺らぎ** | モンロー主義的傾向（東半球への関与低下懸念） |

## 高須幹弥氏の投資見解

### 金についての評価

- **短期**：12%下落しても慌てて売る必要はない
- **長期**：上昇トレンドは継続すると予想
- **超長期**：歴史的にはインフレ率程度しか上昇しない
- **リスク**：金にもボラティリティがある（例：1979年ソ連アフガン侵攻後の急騰→急落→20年ボックス相場）

### 日本株優位の予想

- **今後5-10年は日本株の方が利回りが高い**と予想
- 理由：
  - アメリカ株は割高すぎる（世界中の富が集中）
  - AI革命も既に織り込み済みの可能性
  - 世界のマネーが日本株に流入する見込み

### 自身のポートフォリオ方針

- 現在は「日本株信者」
- 余剰資金があれば日本株を購入
- 金はポートフォリオに少量保有するも、積極的には買わない

## 金価格の歴史的変動パターン

```
1979年：ソ連アフガン侵攻 → 急騰 → 急落 → 20年間ボックス相場
    ↓
リーマンショック → 世界的金融緩和 → 金価格上昇 → 下落
    ↓
コロナショック → 世界的金融緩和 → ロシアのウクライナ侵攻
    ↓
ロシアのドル資産凍結 → 中央銀行の金買い増し → 現在の急騰
    ↓
トランプ発言（FRB議長指名） → 12%下落（今回）
```

## 結論

- 金価格の12%下落は一時的な調整であり、長期的な上昇トレンドは継続する可能性が高い
- ただし、超長期で見れば株式の方がパフォーマンスが良い傾向がある
- 今後5-10年のスパンでは、日本株がアメリカ株や金を上回る可能性がある
- ポートフォリオに金を入れるのは資産保全目的として有効だが、金にもリスクがあることを認識すべき

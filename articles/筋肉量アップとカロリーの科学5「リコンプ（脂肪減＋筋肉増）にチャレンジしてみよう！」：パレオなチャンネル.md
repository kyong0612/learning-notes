## 筋肉量アップとカロリーの科学#5「リコンプ（脂肪減＋筋肉増）にチャレンジしてみよう！」

「筋肉量アップとカロリーの科学」の続きです！([#1](https://ch.nicovideo.jp/paleo/blomaga/ar2224587),[#2](https://ch.nicovideo.jp/paleo/blomaga/ar2224660),[#3](https://ch.nicovideo.jp/paleo/blomaga/ar2224786),[#4](https://ch.nicovideo.jp/paleo/blomaga/ar2225068))

このシリーズでは、「 **筋肉を増やすための正しいカロリー設定とは？** 」って問題を掘り下げております。筋肉を増やすためには絶対に適切なカロリー摂取が必要なんだけど、現時点では「筋肉を増やしたいならカロリーは大量に摂れ！」や「筋肉を増やすためにはいつも通りのカロリーでいい！」みたいに真逆の見解が出てまして、いったいどうすればいいのかよく分かんないんですよね。

前回は「追加カロリーの設定法」を掘り下げてみましたんで、今回は誰もが気になる **「リコンプができる人とは？」** という問題に取り組んでみましょう。

## リコンプ（脂肪減＋筋肉増）は実在するのか？

リコンプについては今までも軽く触れてきましたが、あらためて定義をチェックしておくと、

- **脂肪を落としながら筋肉を増やす**

って現象のことを意味しております。「食べながら痩せて筋肉までつく！」ってのは筋トレ好きには夢のような変化ですが、これを「リコンプ（Body Recomposition）」と呼んでるんですな。

「そんな上手い話があるのかねぇ」と思いたくなる話ですが、実はこのような相反する変化ってのは意外と起きることが、近年の研究で何度か確認され始めてるんですよ(R)。リコンプってのは決して与太話じゃないわけですね。

が、当然ながら、リコンプってのは誰でもできるわけではなく、実現するにはいくつかの“条件”を乗り越える必要があるので注意しましょう。ここを勘違いすると、ただの「停滞した人」になっちゃうんで。

ということで今回は、リコンプの研究をベースに、

- リコンプできる条件
- 実際に成功していた事例
- あなたがリコンプに適しているかどうかを判断する方法

などをまとめていきます。こいつは体をデカくするためには欠かせない知識なので、要チェックであります。

## 筋トレ経験1年以上でもリコンプできる人は何が違うのか？

リコンプについて考えるうえで、まず紹介したいのがタンパ大学のバラカット先生による研究([R](https://journals.lww.com/nsca-scj/fulltext/2020/10000/body_recomposition__can_trained_individuals_build.3.aspx))であります。この論文は、 **筋トレ経験1年以上の中級者** を対象にした10本の研究をピックアップしたレビューでして、そのうちほとんどで「実験の参加者にリコンプが起きていた」という結果を報告してるんですよ。こいつは面白いですねぇ。

リコンプした人たちの平均を見てみると、ざっくりこんな感じになります。

- **減った脂肪量** ：−0.1〜−2.7kg
- **増えた筋肉量** ：＋0.6〜＋5.0kg

ご覧のとおり、見事に脂肪を減らしつつ筋肉が増えてまして、一部の被験者では **たった1日28kcalの軽い追加カロリー** でも筋肉が増え、脂肪が減ったというパターンも見られたらしい。つまり、昔の常識であった「リコンプなんて初心者しか無理！」という説が揺らいできたよーって話でして、これはなかなかすごいもんですな。

では、リコンプが起きる人には、どのような条件が備わっているのか？　バラカット先生がピックアップした10本の研究から見えてくる成功パターンをまとめると、以下のようになります。

### 成功パターン① 週3回以上の「進歩的な筋トレ」

筋肥大の基本原則のひとつに「漸進的過負荷」ってのがあります。これは過去にも述べたポイントなので、詳しくは「 [筋トレにおいてボリュームの次に大事な「プログレッシブ・オーバーロード」のガイドライン](https://daigovideolab.jp/blog/1859692) 」をご覧ください。

簡単に言えば、これは「筋肉にかかる負荷を少しずつ上げ続けることで、成長の刺激を与え続ける」って手法のことでして、リコンプってのは“エネルギーが限られている状態”で筋肉を増やす作業なので、 **刺激が弱いと成長が起きなく** なっちゃうんですよね。実際、バラカット先生が集めた成功例のほとんどで、「負荷を上げていくこと」が明記されてますからねぇ。つまり、「週3で適当に筋トレしてたら勝手にリコンプできる」みたいな甘い話ではないってことですな。

ここでの実践ポイントは、以下のようになります。

- **トレーニング記録をつける** ：筋トレを“なんとなく”やってると、成長が停滞しても気づけないので、毎回「何kgで何回やったか」「RIR（限界からの余裕）」をアプリやノートで記録するのが吉。 筋トレは再現実験なので、「前回よりどれだけ良かったか」がわからなければ、進歩してるかもわからなくなりますからね。私は「Strong」ってアプリを使ってますが、他にもFitNotesやGoogleスプレッドシートなど、好きなものを選んでください。
- **毎週「重量・回数・セット数」のどれかを1つでも伸ばす** ：いわゆる「1％ルール」ってやつで、すべてを一気に伸ばす必要はなく、 たとえば「前回50kg×8回→今回は9回」でも立派な進歩だと言えます。こういった小さな進化を積み重ねることで筋肉に「環境の変化」は伝えられるので、週1で1種目だけでも記録更新できていればOKだとお考えください。「今週はスクワットだけ＋1回できた！」でもリコンプ的には十分なんで。
- **RIR（Reps in Reserve）を目安に、ギリギリ残すくらい** **を狙う（RIR1〜2）** ：RIRってのは「あと何回ぐらい筋トレできる余力があるか？」って指標のこと。 リコンプを狙いたいときは、RIRが3以上になると刺激が弱すぎて筋合成が起きにくくなります。 逆にRIR0（限界まで）でやり続けると回復が追いつかず逆効果になりかねないので、ベストは「RIR1〜2」＝もう1〜2回できそうな強度になります。ここが無理なく追い込める“成長ゾーン”なので、くれぐれも「RIR1〜2」の範囲に収めるようにしましょう。

### 成功パターン② 除脂肪体重（≒筋肉）あたり「2.6〜3.5g/kg」の高タンパク

リコンプは、カロリー収支がプラスマイナスゼロに近い状態で起きる現象でして、このときに **タンパク質が足りていないと絶対に筋肉は増えません** 。バラカット先生のレビューでは、 **2.6〜3.5g/kgの除脂肪体重** という非常に高い摂取量が共通していたそうでして、たとえば、

- 体重60kgで体脂肪20%の人は除脂肪体重が48kgになるので、2.6〜3.5g × 48kgで1日 **125〜168gのタンパク質が必要**

って計算になります。こりゃあかなりの量ですなぁ。

ここでの実践ポイントは、以下のようになります。

- **毎食に 30〜40g ずつ分割摂取** ：プロテインで最も大事なのは摂取量でして、普通に筋肉を増やしたいだけなら、1回あたりの摂取量やタイミングにこだわる必要はなし。ただし、リコンプを目指すとなると、さすがに摂取すべき量が多すぎるので、こまめに分けて摂取したほうが、体内でたんぱく質が効率的に使われて良い感じです。たとえば、 **朝30g、昼40g、夜40g、間食20g** ぐらいのペースでこまめに分けて、たんぱく質を摂取するわけですね。特に夜は寝ている間の合成に使われるので、ちょっと多めに摂取しとくのがお勧めです。
- **プロテインパウダー（ホエイ、カゼイン）を併用して達成しやすくする** ：高タンパク食を「自然食品のみ」で達成しようとすると、食事量が多くなりすぎたり、脂質やカロリーが過剰になりやすかったり、毎日メニューを考えるのが大変だったり、という問題が起こるわけです。その点でプロテインパウダーを使うのは必須なんですが、ここで大事なのはプロテインの使い分けでして、筋トレの後にはホエイプロテインを1スクープ（20〜25g）、寝る前にはカゼインを1スクープ（20g前後）みたいな感じで使い分けるのもありでしょう。ホエイプロテインは吸収が速くて筋肉の成長をすぐ刺激してくれて、トレーニング後か朝に摂取するのがお勧めです。一方で、カゼインプロテインは吸収がゆっくりなので、長時間の筋肉合成を維持するのに最適で、寝る前か間食に使うと良いでしょう。
- **消化に自信がない場合は、脂質の多い動物性タンパク質を避ける** ：肉類などの「高タンパク食材」は同時に脂質も多く含むことが多いのが難点（特に加工肉・豚肉・牛肉）。脂質は消化時間が長く、胃腸に負担をかけやすいので胃もたれを起こしやすく、そのせいで食事回数が減り、結果的にタンパク質が不足しやすいんですよ。なので、リコンプを行う際は、鶏むね肉（皮なし）、白身魚（タラ・カレイなど）、ギリシャヨーグルト、豆腐・納豆・おからなどを中心に回すのが大事になります。また、肉は茹でる・蒸す・スープ化して消化を助け、小分けにして回数を増やすのも良い対策になります（1日4〜5食）。

### ③ 睡眠「7〜9時間」

#### 睡眠中に分泌される成長ホルモンと筋タンパク合成（MPS）の活性化は、リコンプ成功のカギと言えるレベルで大事なポイント。私たちの体内には概日リズムと呼ばれる“体内時計”が存在してまして、これは筋タンパク合成や成長ホルモンの分泌と密接に関係しているので、就寝・起床の時間がバラバラだと、このリズムが乱れて成長ホルモンの分泌ピークがズレ、MPS（筋タンパク合成）がうまく働かなくなっちゃうんですな

実際、過去の研究では **たった1〜2時間の睡眠不足でも筋肉合成が低下** することが報告されてますんで、睡眠をないがしろにしたらリコンプは失敗するものとお考えください。睡眠の改善については、昔のパレオチャンネルでもさんざんやってるので、ここでは繰り返しません。とにかく、「毎日同じ時間に寝て、同じ時間に起きる」って睡眠リズムの固定にこだわり、リコンプ中の睡眠には特にこだわってください。

### ④ ストレス管理

#### 慢性的なストレスは、コルチゾールの分泌を増やして筋肉の合成を阻害する作用を持っております。また、ストレスによって睡眠が乱れると、それで筋トレの質が下がり、食欲も乱れる…という負のスパイラルにハマっちゃうんですよね

ということで、ここでの実践ポイントは、以下のようになります。

- **瞑想（NSDR、マインドフルネス）を1日10〜15分** ：瞑想は、ストレスホルモンであるコルチゾールの分泌を抑えると同時に、自律神経（交感神経・副交感神経）のバランスを整える効果が知られております。なので、毎度おなじみマインドフルネス瞑想をやってみるもよし、それが面倒ならNSDR（Non-Sleep Deep Rest）みたいに、横になって行う“浅い仮眠”のような休息法を使うのもいいでしょう。この手法については、YouTubeやアプリで「NSDR」で検索すれば無料ガイド音声が豊富に出てきますんで、寝る前か昼休憩にするのがおすすめ。
- **ゆっくりした有酸素運動（散歩や軽いサイクリング）を週に数回** ：「運動がストレス解消になる」のは有名なんだけど、高強度すぎる運動は逆にストレス負荷になる可能性もあるのが難点。 一方で、軽い有酸素運動（低〜中強度）は、脳内のセロトニンやドーパミンを活性化させ、ストレスを緩やかに鎮める働きがありますんで、こちらを取り入れておきましょう。具体的には、朝日を浴びながらの20〜30分散歩、ゆっくりとペダルをこぐサイクリングなどで十分であります。心拍数の目安としては、軽く息が上がる程度（最大心拍数の50〜60％）を目指しましょう。
- **SNSやニュースから距離を置く（情報デトックス）** ：情報過多は、知らぬ間に脳を過剰に興奮状態にさせ、コルチゾールの分泌を促進しちゃうので、リコンプの邪魔になってしまうわけです。 特に現代では、SNSが「比較・炎上・刺激」のオンパレードであり、脳の“闘争・逃走反応”を慢性的に誘発するんですよ。実際のところ、研究では「ストレスが高いとリコンプ成功率が下がる」ことが示されてまして、軽視できない要素であります。なので、デジタルデトックスは筋肉を育てるためにも必須。

### ⑤ 最低でも「3〜6ヶ月以上」のスパンで評価する

リコンプは変化の速度が遅いため、短期的には体重も見た目もほぼ変わらないことが多め。そのため、

- 「体重が減らない」＝失敗
- 「腹筋が見えない」＝意味ない

と早々に諦めてしまう人も多かったりするんですよね。しかし、これはたいてい評価タイミングが早すぎるだけなので、ここで諦めるともったいなくなっちゃうんですよ。なので、ここでお勧めなのが最低でも「3〜6ヶ月以上」のスパンで評価することでして、実践ポイントは以下のようになります。

- **「1ヶ月で体重が変わらなくても、鏡と筋力で評価」する** ：多くの人は「体重が減った＝成功」「体重が変わらない＝失敗」と思いがちなんだけど、リコンプにおいては体重変化がゼロでも大成功ということが珍しくないんですよ。たとえば、「筋肉＋1.5kg／脂肪−1.5kg → 体重±0kg」みたいな感じで、体重が変わってなくても見た目は引き締まり、筋力もアップしているわけです。なので、毎朝自分の体を鏡で見て「体が引き締まったか？」を調べつつ、さらに筋力の指標として「メイン種目の使用重量」を記録（ベンチ・スクワットなど）し、週単位で「扱う重量が増えているか？」を見ることで、“筋肉がついているか”を推測しましょう。
- **月1回は正面・側面・背面の写真記録を取る（照明条件を固定）** ：「見た目の変化」は、日々の生活の中では自分では気づきにくいもの。それを客観的に捉える手段が「定点写真」でして、特にリコンプでは体重が変わらないことが多いため、“視覚情報”が唯一の評価軸になることがよくあるんですよね。なので、月に1回でいいんで、必ず同じ場所・時間・照明条件・カメラ距離で撮影するようにしてくださいませ（スマホ固定推奨）。その際は、「正面・側面・背面」の3方向を撮っておきましょう。比較アプリ（Progression, BodyTracker）などを使うと、2枚の写真を重ねて確認できてお勧めです。
- **トレーニングログに加えて「見た目メモ」を残す** ：数字や写真に表れない変化（姿勢、肌ツヤ、筋肉の張り、ウエスト感覚など）も、肉体の「進化」を教えてくれる重要な情報のひとつ。こうした主観的な変化はモチベーションの維持に直結しますんで、「腹回りがすっきりしてきた」「二の腕のラインがはっきりしてきた」「Tシャツの肩まわりがキツくなった気がする」「脚の間に隙間が出てきた」のような見た目の変化メモを、週1回の振り返りとしてメモ帳やアプリに残すようにしておきましょう。

## 自分がリコンプ可能かどうかをチェックしてみよう

リコンプは誰でもできるわけではなく、いくつかの条件を満たす必要があります。先ほど紹介した5つの成功パターンを見ると、確かに多くの要素が絡み合っていることがわかりますが、実際に自分がリコンプを狙えるタイプなのか、それともまずは増量や減量のフェーズ分けを考えるべきなのか、判断に迷うこともあるでしょう。

そこで、あなた自身が「リコンプを狙えるタイプかどうか」を簡単に判定するためのチェックリストを用意しました。以下の質問をチェックして、自分がいくつ当てはまるかを考えてみてください。

| 質問項目 | はい / いいえ |
| --- | --- |
| ① 筋トレ歴は1年未満である | はい / いいえ |
| ② 体脂肪率はやや高め（男性15%以上・女性25%以上） | はい / いいえ |
| ③ 過去にトレ経験があるが、いまはブランク明け | はい / いいえ |
| ④ 睡眠が毎晩7時間以上取れている | はい / いいえ |
| ⑤ タンパク質を体重×2.6g以上、毎日摂っている | はい / いいえ |
| ⑥ トレーニングは毎回少しずつ強度を上げている | はい / いいえ |
| ⑦ ストレスがコントロールできている | はい / いいえ |

- **「はい」が5つ以上** ：→ **リコンプを狙える確率はかなり高め。** 今すぐプランを立てましょう。
- **「はい」が3〜4つ** ：→ **条件付きリコンプゾーンです** 。まずは、リストのなかから不足している部分を強化しましょう。
- **「はい」が2つ以下** ：→ **素直に追加カロリーを摂取すべき段階です。** 増量 → 減量のフェーズ分けを考えてみてください。

## まとめ

ということで今回は、デカい肉体を作るために欠かせない「リコンプ」のお話でした。リコンプってのは、決して一発逆転の魔法ではなく「地味だけど正しくやれば起こる現象」だってのがおわかりいただけたのではないかと。

リコンプに挑戦している時は、「体重が変わらないけど…大丈夫？」「服のサイズも変わらないし、意味あるの？」みたいに思うことが多いですが、正しくやれば確実に変化は起きますので諦めないのが肝要であります。かつてはリコンプは初心者だけの特権と思われてましたが、条件が整えば中級者でも脂肪は落ちて筋肉は増えることがわかってきたのはうれしいポイントっすね。ではまた！

## 概要

本動画は、Apple社のDAW「**Logic Pro 11**」の基本操作から楽曲完成までを、実際にポップス楽曲を制作しながら解説する約2時間3分の包括的なチュートリアルです。

### 完成楽曲の特徴

- アコースティックピアノ、ギター、シンセサイザーなど多彩な音色を使用したポップス楽曲
- 女性ボーカル
- ボーカル・ギターの生演奏レコーディングを含む

---

## 1. インストール・初期設定（01:13～）

### Logic Proのインストール

1. App StoreでLogic Proを検索・購入・ダウンロード
2. アプリケーションフォルダから起動
3. 初回起動時に「**完全モード**」を選択（全機能使用可能）
4. 頻繁に使用する場合はDockに保存推奨

### オーディオ設定

- **Logic Pro** > **設定** > **オーディオ**
- **出力デバイス**: 音を出す先（内蔵スピーカー/オーディオインターフェイス/イヤホン）
- **入力デバイス**: マイクや楽器の入力元

### サウンドライブラリのインストール

- **Logic Pro** > **サウンドライブラリ** > **サウンドライブラリマネージャー**
- 「インストールされていないコンテンツを全て選択」→インストール
- ⚠️ 容量が大きいため時間がかかる場合あり

### プロジェクトの作成

- 「**空のプロジェクト**」を選択
- **サンプルレート**: 48kHz推奨（後から変更すると音質劣化の可能性）

---

## 2. ドラム・パターンエディター（03:56～）

### トラックの概念

- **トラック**: 音や演奏を記録するための「列」
- 異なる音や楽器を配置し、重ねて楽曲を作成

### パターントラックの作成

1. 新規トラック作成画面から「**パターントラック**」を選択
2. トラック上で右クリック → 「**パターンリージョン作成**」
3. リージョンにドラムパターンを入力

### 画面操作

| 操作 | 方法 |
|------|------|
| 横方向の拡大縮小 | Shift + Option + マウスホイール |
| 縦方向の拡大縮小 | Option + マウスホイール |
| 再生/停止 | スペースキー |
| メトロノーム | Kキー |

### パターンエディターでの入力

- **ステップ**（マス目）をクリックして音を入力/削除
- 各楽器アイコンをクリックでサウンド確認
- **キック**: 小節の頭
- **スネア**: ハーフビートの場合は3拍目

### サイクル再生

- ルーラー上部のサイクル範囲をクリックで有効化
- **Cキー**でも切り替え可能
- 両端をドラッグで範囲変更

### ハイハットパターン

- **リピートモード**: ステップを上下にドラッグで音を分割（ロール作成）
- 3連や6連などの変化を加えるには:
  1. Command + Option + ドラッグでステップを結合
  2. リピートで分割設定

### ステップ数の変更

- 16ステップ → 32ステップで2小節分のパターンに

### ベロシティの調整

- 編集モードを「**ベロシティ**」に変更
- ステップごとに値を調整してリズムに強弱を付ける
- 周期的な変化で「ノリ」を演出

### 音色の変更

1. ライブラリーからドラムキット全体のプリセットを選択
2. 個別の楽器（スネアなど）を選択してライブラリーから差し替え可能

## 3. ピアノ・ピアノロール（13:44～）

### プロジェクトの保存

- **Command + S**
- 「次のファイルをプロジェクトにコピー」: 全てにチェック推奨

### MIDIトラックの作成

1. 左上の「**+**」ボタン → **ソフトウェア音源**
2. 音源リストから「**スタジオピアノ**」を選択
3. 右クリック → 「**MIDIリージョン作成**」

### コード進行の入力

- 使用コード例: **Fmaj7 → E7 → Am7 → C**（Aマイナーキー）

### リージョンの調整・ループ

- リージョンの終端をドラッグして長さを調整
- **Lキー**: 楽曲終わりまで自動反復
- 終端の「ループポインター」をドラッグ: 特定の長さで繰り返し

### ピアノロールでの入力

- ダブルクリックでピアノロールを表示
- **鉛筆ツール**: 音符を入力
- **ポインターツール**: 音符の移動・長さ調整

### ディビジョンの変更

- コントロールバーをカスタマイズ → 「**拍子ディビジョン**」にチェック
- 1/4（四分音符）～1/16（十六分音符）で増目の細かさを変更

### Command + クリックツール

- ツールメニュー隣から設定
- Command押下時のみ別ツール（鉛筆など）に切り替え
- 誤入力を防ぎつつ効率的に作業

### Undo/Redo

- **Command + Z**: 元に戻す（Undo）
- **Shift + Command + Z**: やり直し（Redo）

---

## 4. ベース・Session Player（19:33～）

### リージョンのループ・複製

- **Lキー**: 楽曲終わりまでループ
- 終端のループポインターをドラッグ: 特定の長さでループ
- **Control + L**: ループを個別リージョンに分割
- **Option + ドラッグ**: 任意の位置にコピー

### キーの変更

- 画面上のキーディスプレイから変更
- 右クリック → 設定表示 → 「キー/プロジェクト終了」にチェック

### コードトラック

1. グローバルトラックを表示
2. コードの列に鉛筆ツールでコードを追加
3. ルートノートとコードの種類を選択
4. リージョン同様にドラッグで長さ調整

### コードのグループ化

- Shiftキーを押しながら複数選択
- 右クリック → 「**コードをグループ化**」
- **Command + C / V**: コピー＆ペースト

### Session Player（Bass Player）

1. 新規トラック → **Session Player** → **Bass Player**
2. 自動で8小節分のリージョンが作成
3. キー・コード進行に沿って自然な演奏を生成

### Bass Playerの調整

| パラメータ | 効果 |
|-----------|------|
| スタイル | フレーズのタイプ（R&B、Funkなど） |
| 強さ | 演奏のタッチ |
| 複雑さ | フレーズの音数 |
| フレーズの再生成 | 同じ設定で異なるバリエーションを生成 |

### 音量バランスの調整

- 各トラックのスライダーで調整
- 一度下げてから楽曲を流しながら調整

## 5. メロディー MIDIレコーディング（26:52～）

### 音色を選ぶ

1. ソフトウェア音源トラックを作成（空のチャンネルストリップ）
2. ライブラリーから音色を選択
3. 検索ボックスでキーワード検索も可能

### MIDIレコーディング

1. **録音可能ボタン**を点灯
2. **録音ボタン**をクリック（4拍カウント後に開始）
3. カウント数は右クリックで変更可能

### サイクル録音とテイク

- サイクル範囲を設定して繰り返し録音
- **Logic Pro** > **設定** > **録音** → サイクロンを「**テイクフォルダーを作成**」に変更
- 矢印をクリックでテイク選択

### クオンタイズ

- タイミングを一括補正する機能

1. ピアノロール左の「**タイムクオンタイズ**」から音符を選択（1/16など）
2. 「強さ」が100であることを確認
3. 全ノートを選択（**Command + A**）
4. **Qボタン**をクリック

### テイクを組み合わせる

- リージョンを分割（**はさみツール**、またはTキーで切り替え）
- 分割後、リージョンごとにテイクを選択

### 録音結果を取り込む

- 即興演奏中に気に入ったフレーズを発見した場合
- **レコーディング取り込み**ボタンをクリック

### リージョンの結合

1. テイクフォルダーの数字をクリック → 「**フラット化**」
2. リージョンを全選択 → 右クリック → 「**バウンスして結合**」（または **Command + J**）

### ベロシティの一括調整

- ノートを選択 → スライダーを操作
- **Shift + Option + スライダー**: 全て同じ値に設定

### トリム

- **編集** > **トリム** > **ノートの終了位置から後続のノートまで**
- ノート間の隙間を自動で埋める

## 6. アレンジ Keyboard Player（37:10～）

### 手動でピッチを変更

- ピアノロールでノートを選択 → ドラッグ
- **Option + 上下矢印**: 半音ずつ移動
- **Shift + Option + 上下矢印**: 1オクターブずつ移動

### 一括でピッチを変更

- リージョンを選択
- インスペクタの「**トランスポーズ**」に値を入力（例: -5で5半音下げ）
- ⚠️ Session Playerリージョンでは使用不可

### 自動でピッチを変更（ピッチソース）

- リージョンの「**ピッチソース**」を設定
- **Session Playerリージョン**: コードトラック
- **MIDIリージョン**: キー

### キー変更時の連動

- キーを変更すると「コードトラックも連動させるか」と確認
- 「**トランスポーズ**」を選択

### Keyboard Player

1. 新規トラック → **Session Player** → **Keyboard Player**
2. コード伴奏を自動生成

### 演奏内容の変更

| 設定 | 効果 |
|------|------|
| タイプ（ブロック構図など） | 伴奏スタイル |
| リズムパターン番号 | 具体的なリズム |
| 8分音符を継続 | リズムの隙間を埋める |
| コード | コード進行切替に連動 |
| トラック | 他トラックとリズムを揃える |
| 両手スライダー | 音域の調整 |
| フィルの量/複雑さ | 変化の頻度と複雑さ |

### 分割して設定する

- はさみツールでリージョンを分割
- 各リージョンで異なる設定が可能

### MIDIリージョンに変換

- 右クリック → **変換** → **MIDIリージョンに変換**
- 手動でノートの編集が可能に

### 音色を変更する

- インストルメントの右端からリストを表示
- 「**クイックサンプラー**」など別の音源に変更可能
- エフェクトのオン/オフで音色を調整

## 7. ギターの録音・コンピング Flex Time（48:52～）

### オーディオ設定

- **Logic Pro** > **設定** > **オーディオ** > **デバイス**
- **入力デバイス**: ギターが接続されたオーディオインターフェイス
- **I/Oバッファサイズ**: 64推奨（遅延軽減）、パソコン負荷高い場合は数値を上げる

### ギタートラックの作成

1. **オーディオトラック** → **ギターまたはベース**
2. **オーディオ入力**: ギターが接続されているチャンネルを指定
3. **入力モニタリング**ボタンを有効化

### 音色を変える

- ライブラリーから **Electric Guitar & Bass** のプリセットを選択
- アンプやペダルボードで細かい音作りも可能

### レコーディング

1. **録音可能ボタン**を有効化
2. 録音ボタンまたは **Rキー** で開始
3. スペースキーで停止

### テイクコンピング

1. リージョン左隣のマークが「山」のようになっていることを確認
2. メインのテイクを選択
3. 差し替えたい箇所を範囲選択
4. より良いテイクを選択

### テイクを複製する

- 「A」をクリック → 「**コンプを複製**」
- 異なるバージョンを比較可能

### フラット化

- メニューから「**フラット化**」または「**フラット化して結合**」
- テイク選択を確定

### パンチイン（オートパンチ）

1. コントロールバーをカスタマイズ → 「**オートパンチ**」有効化
2. 赤い範囲を取り直したい箇所に設定
3. どこから録音開始してもその範囲のみ記録

### フェード

- **フェードツール**でリージョン端をドラッグ
- 減衰カーブを描いて余韻を調整

### Flex Time

1. エディター内の **Flex** ボタンを有効化
2. **トランジェント**（点線）をクリック → **フレックスマーカー**（実線）に変更
3. マーカーを左右にドラッグでタイミング調整
4. 前後にもマーカーを作成して他の音が移動しないように

## 8. Apple Loops（57:33～）

### ループブラウザ

- ループブラウザボタンをクリック
- MIDI/パターン/オーディオリージョンのループが表示
- 選択して試聴、ドラッグ＆ドロップで配置

### 検索条件を絞り込む

| カテゴリ | 設定例 |
|----------|--------|
| リージョンタイプ | オーディオ |
| 音源 | パーカッション |
| ジャンル | ワールド |
| 詳細 | 暗い |

### フィルを作る

- 同じ名前の連番ループを切り替えてバリエーション
- はさみツールで分割して変化をつける

### 音源を差し替える

- MIDIトラックのインストルメントを変更
- 例: ES1（シンセサイザー音源）

### パンニング

- インスペクタの**パンノブ**をドラッグ
- 左右に配置してステレオの広がりを演出

## 9. 高度なテクニック（01:04:50～）

### ドラムフィルの追加

1. フィルを入れたい箇所でリージョンをカット
2. **Session Player** → **Drummer**（エレクトロ/トラップ）を追加
3. 楽器アイコンを全てオフ → フィルのみ残す
4. 「フィルの量」「フィルの複雑さ」を調整
5. 「再度実行」でバリエーション作成
6. MIDIリージョンに変換して固定

### Track Stack

1. 複数トラックを **Command + クリック** で選択
2. 右クリック → 「**トラックスタックを作成**」
3. 「**フォルダースタック**」を選択
4. まとめてソロ/ミュートが可能

### ミュートツール

- ツールメニューから**ミュートツール**を選択
- リージョンやノートをクリックで一時的にミュート
- 再度クリックで解除

### クラッシュシンバル（アクセント）

1. MIDIトラック → 空のチャンネルストリップ
2. ライブラリーから **エレクトロドラムキット** > **キットピーセス** > **シンバルズ**
3. C3が基準ピッチ、上下で高さ変更可能
4. セクションの始まりやドラムフィル後に配置

### リバースシンバル

1. MIDIリージョンを選択
2. 右クリック → **バウンスして結合** → **所定の場所にバウンス**
3. 「ファイルにオーディオテールを含める」にチェック
4. オーディオリージョンを選択 → 右クリック → **再生** → **逆再生をオン/オフ**
5. セクション終わりに配置して盛り上がりを演出

### ストリングス

1. MIDIトラック → **スタジオストリングス**
2. ストリングアンサンブルを選択
3. コード伴奏をコピー

### MIDI FX（アルペジエーター）

1. インスペクタの **MIDI FX** をクリック
2. 「**アルペジエーター**」を選択
3. **レート**: フレーズの細かさ
4. **パターン**: 音の高さの並び

### MIDI FXを書き出す

1. ストリングストラックを複製
2. 複製したトラックのインスペクタ → **内蔵MIDI入力** → 元のトラックを選択
3. 録音するとMIDI FX適用後の演奏が記録される

### アーティキュレーション

1. インスペクタ → **アーティキュレーション** から音源とプリセットを選択
2. ピアノロールに「アーティキュレーション」項目が追加
3. ノートを選択して奏法（スタッカートなど）を切り替え

### サスティンペダル

1. オートメーション表示ボタンを点灯
2. 「ノート/ベロシティ」を「**サスティン**」に切り替え
3. ペダルを踏まない箇所: 0
4. ペダルを踏む箇所: 最大値
5. コードが変わる節目ではペダルを離す

## 10. ボーカルの録音（01:19:22～）

### レコーディング準備

1. 前奏を用意（キーやテンポ確認用）
2. オーディオトラックを作成
3. 入力設定でマイクのチャンネルを指定
4. 入力モニタリングを有効化
5. 入力デバイスを確認（オーディオインターフェイス）

### フリーズ

- CPU負荷が高い場合に使用
- トラックを右クリック → **トラックヘッダーを設定** → **フリーズ**にチェック
- フリーズボタンを有効にして再生で処理実行
- ⚠️ フリーズ中は音源・エフェクト・演奏内容の変更不可

### ボリューム・リバーブ設定

- オケのトラックを全選択 → フェーダーを一括で下げる
- ボーカルトラックにリバーブ（**スペースデザイナー**）を追加
- **ボーカルプレート**プリセットを選択
- **Wet**でリバーブレベルを調整

### オートパンチ

- 部分的に取り直す場合に使用
- **マーキーツール**で範囲選択 → オートパンチの範囲が連動

### ハモリのレコーディング

- 別トラックを作成
- メインボーカルの音量を下げ、パンを片側に振る
- タイミングやニュアンスを確認しながら録音
- ダブリング（同じパートを2回録音して左右に配置）でステレオ感を演出

---

## 11. Flex Pitch オートメーション（01:28:33～）

### Flex Pitch

1. リージョンをダブルクリック → エディター内の **Flex** ボタンを有効化
2. **フレックスモード** を「**Flex Pitch**」に変更
3. ピッチとタイミングがバーで表示

### ピッチ補正

| 操作 | 効果 |
|------|------|
| ノートをドラッグ | 半音単位で移動 |
| 上側中央のドット | ファインピッチ（平均的な高さ） |
| 下側中央のドット | ビブラート（0で真っすぐ） |
| 上側両サイドのドット | 始点/終点のピッチ |
| はさみツール | ノートを分割 |

### ピッチ補正のコツ

- ファインピッチは0を目指す
- ピッチのラインが中央に長く滞在するように調整
- 過剰な補正は「ケロリ」（音質劣化）を招く
- 音の入り口は演奏ニュアンスが多いため、不自然でなければ調整を控える

### タイミング補正

- ノートの両端にカーソル → 表示が切り替わったらドラッグ
- 隣接するノートの波形も変化するため注意
- 直接リージョンをカットして配置をずらす方法もあり

### 無音部分を削除

- リージョンを選択 → **機能** → **オーディオリージョンから無音部分を削除**
- **Control + X** でも呼び出し可能
- 閾値を調整して不要な部分を自動削除

### オートメーション

1. **オートメーション表示ボタン**を点灯（または **Aキー**）
2. 「リード」を「**タッチ**」に切り替え
3. 曲を再生しながらパラメーターを操作
4. 操作内容が書き込まれる

### オートメーションモード

| モード | 動作 |
|--------|------|
| リード | 読み込みのみ（書き込みなし） |
| タッチ | パラメーターに触れている間だけ書き込み |
| ラッチ | パラメーターに触れた後も書き込み継続 |
| ライト | 再生開始から全て上書き |

## 12. ミキシング前編（01:41:57～）

### ミキサー

- **ミキサーボタン**または **Xキー** で表示
- 表示設定: 「**トラック**」を選択（必要なチャンネルのみ表示）

### ショートカットの割り当て

- **Logic Pro** > **キーコマンド** > **割り当てを編集**
- 「ミキサーの取り消し」などにショートカットを割り当て可能

### 複数の出力があるトラック

- ドラムキットなど個別チャンネルが含まれる場合
- 下の矢印をクリックで展開
- 経由チャンネル（矢印付き）の音量を0に設定 → 個別チャンネルのみで調整
- Track Stack用のフェーダーも初期値に戻す

### 1つずつ楽器を加える

1. **キック**: -10dB程度
2. **ベース**: キックより少し小さく
3. **コード伴奏**（ピアノ）: キック・ベースが聞こえる程度
4. **ボーカル**: 歌詞が明瞭に聞こえる大きさ
5. 以降、聞かせたい順番で加える

### ミキサーの操作

- **Shift + クリック**: 複数チャンネル選択（連動操作）
- **Command + クリック**: 飛び飛びで選択
- チャンネルをドラッグ: 位置を移動
- **Control + ソロボタン**: セーフソロ（常に鳴らす）

### EQ（イコライザー）

- チャンネル上部の **EQ** をクリック → **チャンネルEQ** が立ち上がる
- 左側: 低音域、右側: 高音域
- プリセットから選択（ボーカル、ドラム、キーボードなど）
- EQ適用後は音量が変わるため、ピークメーターを確認して調整

## 13. ミキシング後編 マスタリング・書き出し（01:54:45～）

### 音量のムラを減らす（ボリュームオートメーション）

1. **オートメーション表示ボタン**をクリック（または **Aキー**）
2. 編集対象が「ボリューム」であることを確認
3. 曲を再生して音量が小さい/大きい部分を探す
4. 編集点を追加してカーブを描く
5. ⚠️ 垂直なカーブはノイズの原因、斜めに描く

### VCA（ボリュームオートメーション後の全体調整）

- 空きスペースで右クリック → 「**新規VCAを作成**」
- 接続チャンネルの相対的な音量を変更可能

### リバーブをセンドでかける

1. チャンネルの **センド** セクションをクリック → 未使用のバスを選択
2. 新しく **Aux** チャンネルが作成される
3. Auxにリバーブ（スペースデザイナーなど）を追加
4. **Dry**: 下げる、**Wet**: 上げる
5. センドのノブでリバーブ量を調整
6. 複数チャンネルで同じバスを選択 → リバーブを共有

### マスタリング

1. **ステレオアウト**チャンネルの「**マスタリング**」をクリック
2. 自動でマスタリング処理が実行

### マスタリングの調整

| パラメータ | 効果 |
|-----------|------|
| オートEQ | 音質変化の強さ |
| ラウドネス | 音量感 |
| エキサイト | 中高音を強調して明るく |
| ワイドネス | ステレオ感の広がり |

### バウンス前の準備

1. 書き出し範囲を設定（サイクル範囲）
2. スナップをオフにして細かく調整
3. ステレオアウトチャンネルのボリュームオートメーションでフェードイン/アウト

### 楽曲をバウンスする

1. サイクルを有効化
2. **ファイル** > **バウンス** > **プロジェクトまたは選択範囲**
3. 設定:
   - 用途: **PCM**
   - ファイルタイプ: **Wave**
   - ビット数: **24ビット**
   - サンプルレート: **48kHz**
   - フォーマット: **インターリーブ**
   - ディザリング: **なし**
4. 保存場所とファイル名を設定 → **バウンス**

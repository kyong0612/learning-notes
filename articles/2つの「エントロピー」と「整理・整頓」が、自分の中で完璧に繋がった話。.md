# 2つの「エントロピー」と「整理・整頓」が、自分の中で完璧に繋がった話。

ref: <https://x.com/tanashin115/status/2032637751993184761>

---

## 導入：きっかけとなったPodcast

* Podcast「**オブ狂い**」の「#10 情報の洪水に溺れないための3つの方法」で紹介された「**エントロピーを下げる**」という考え方が出発点。
* 10X情報処理エキスパート講座で「エントロピーを下げよ！」という概念を学んでいたことが、Podcastへの反応を高めた。
* Podcastでは「**エントロピーには2つの種類がある**」「**その2種類は、それぞれ整理と整頓という行動に結びつく**」と語られていた。

## 「整頓は整理ではない」— 加藤秀俊『整理学』（1963年）

* 加藤秀俊氏の著書「**整理学 忙しさからの開放**」（1963年）が、「整理」と「整頓」の違いを明確に提示。

![Image](https://pbs.twimg.com/media/HDVgEfzbAAA_kdE?format=png&name=large)

* **整頓**: 散らばっているものを目障りにならないように綺麗に片付けること（見た目の問題）。
* **整理**: **「必要なものが必要なときに取り出せるようになっている状態」**（本質的な機能の問題）。

> 整頓は整理ではない。逆にいえば、見た目には乱雑な整理もありうるということである。見た目には散らかっているようでも、右手のほうにはやりかけの仕事Aに関する書類、まん中のはBに関する資料、左手のはC関係の報告書・・・、といったふうな「分類」がおこなわれているのが整理ということである。

* この区別により、「整理がよくて整頓が悪い人」「整頓がよくて整理が悪い人」「両方よい人」が存在することが分かる。
* **アインシュタイン、マーク・トウェイン、スティーブ・ジョブズ**の机は散らかっていたが、頭の中がカオスだったとは考えにくい。

![Image](https://pbs.twimg.com/media/HDVeZnQbUAAw2iY?format=png&name=large)

![Image](https://pbs.twimg.com/media/HDVebkMaMAA-lAK?format=png&name=large)

* **核心**: 物理空間が散らかっていても、頭の中が整理されていれば、創造性は損なわれない。

## 2つのエントロピー

### 1. 熱力学のエントロピー — 「乱雑さの度合い」

* 熱力学第二法則：**エントロピーは増大する** — 放っておけば物事は必ず無秩序・カオスに向かう。
* 具体例：部屋は掃除しなければ汚れる、体は鍛えなければ衰える、組織は放置すれば崩壊する、コーヒーにミルクを入れると混ざる。
* **あらゆるものは努力せずして整った状態を保つことはできない。**

![Image](https://pbs.twimg.com/media/HDVerMraMAMtc2q?format=png&name=large)

* 豆知識：エントロピーの漢字は「**熵**」。火の商 = Q/T（熱を割る）を表す。

### 2. 情報理論のエントロピー — 「予測のしにくさ（不確実性）」

* 1948年、**クロード・シャノン**により定式化。**「平均情報量」＝「予測のしにくさ（不確実性）」** を数値化する概念。
* 2択の場合：
  * **確率100%（または0%）** → 不確実性ゼロ → **エントロピー最小（0）**
  * **確率99%** → 不確実性低い
  * **確率50%** → 「どっちに転ぶか分からない」 → **エントロピー最大（1）**

![Image](https://pbs.twimg.com/media/HDVfFj7aMAccJMS?format=png&name=large)

* **選択肢の数が増えるとエントロピーはさらに増大する**（2択→6択→10択…と候補が増えるほど不確実性が高まる）。
* 情報管理への応用：ファイルの所在が分からず候補が乱立している状態 = 情報エントロピーが高い。タグ・リンク・検索で「探せば必ず見つかる」仕組み = 情報エントロピーが低い。

![Image](https://pbs.twimg.com/media/HDVfLO9aMAENdfG?format=png&name=large)

* 筆者はObsidianに一本化する以前、メモツールが複数に分散しており、情報エントロピーが非常に高かった。

![Image](https://pbs.twimg.com/media/HDVfQfzbIAAp90q?format=png&name=large)

## 核心的発見：2つのエントロピーは「整頓」と「整理」に対応している

| | 熱力学のエントロピー | 情報理論のエントロピー |
|---|---|---|
| **意味** | 物理的な乱雑さの度合い | 不確実性の度合い |
| **下げる行為** | 散らかった部屋を片付け、物を並べ直し、見た目の秩序を保つ | 迷いを消し、必要なものを必要なときに取り出せる仕組みを作る |
| **対応する概念** | **整頓** | **整理** |

![Image](https://pbs.twimg.com/media/HDVfaSYakAA6UNU?format=png&name=large)

* 加藤秀俊氏が半世紀以上前に「整頓は整理ではない」と言い切った話が、2つの異なるエントロピーという**科学的フレームワークで裏付けられている**。
* 見た目が綺麗でも必要なものが取り出せなければ → 情報エントロピーは高い。
* 見た目が散らかっていても配置を把握していれば → 情報エントロピーは低い。
* **アインシュタインの机**: 熱力学のエントロピーは高かったが、情報エントロピーは低かった。

## 応用：「曖昧な指示」は情報エントロピーを爆上げする

### 実体験1 — プロジェクトリーダーBさんの事例

* Bさんの悩み：メンバーが思うように動いてくれない。
* 筆者のアドバイス：**「やって欲しいことがあるなら指示は明確にして出す」** — 目的・タスク・期限・レベル感・予算・参考資料を全て明示する。
* Bさんの気づき：指示が曖昧で「察して考えて」というスタンスになっていた。
* **結果**: 指示内容から曖昧さをなくすことを意識したところ、メンバーの動きが大幅に改善。
* **「あいまいな指示には、人を動けなくしてしまう力がある」**

### 実体験2 — 「仕様に書いてないが普通のエンジニアならできるでしょ」

* Togetterまとめ（[参照](https://togetter.com/li/1860161)）：仕様書に書かれていないことを「普通ならできる」と求められたエンジニアが、作業クオリティが低いと指摘された事例。
* 仕様確認の場なし、中間テストなし、品質確認はテストフェーズまで一切なし。
* 「普通」の基準が不明確 → 受け手の不確実性は最大化 → **情報エントロピーが爆上げされた案件**。

### 理論的裏付け — チップ・ハース教授の「意思決定の麻痺」

![Image](https://pbs.twimg.com/media/HDVf8s_aMAIFZkR?format=jpg&name=large)

* スタンフォード大学のチップ・ハース教授は、選択肢やあいまいさが増えると人が凍りつき動けなくなる現象を「**意思決定の麻痺**」と呼ぶ。
* > 変化を成功させるには、あいまいな目標を具体的な行動に置きかえることが必要だ。
* **人に動いてもらいたいなら、指示から「あいまいさ」を消すこと = 相手の情報エントロピーを下げてあげること = 「整理」そのもの。**

## 結論：2つのエントロピーの両方を下げていく

* 日々の暮らしや仕事は、**2種類のエントロピーとの終わりなき付き合い**。
  * **整頓** = 熱力学のエントロピーを下げる（部屋を掃除、デスクを片付け、物理的秩序を保つ）
  * **整理** = 情報エントロピーを下げる（リンク・タグで取り出せる仕組みを作る、明確な指示で相手の不確実性を減らす）
* どちらのエントロピーも、放っておけば必ずカオスに向かう。
* **意識的に「整頓」と「整理」の両方に取り組むことが重要。** 自分に対しても、他者に対しても。
* スタジオジブリのプロデューサー鈴木敏夫氏の言葉：**「仕事の8割は整理整頓だ」**。2つのエントロピーを下げることが、ジブリ作品のような高密度・高品質な情報（作品）を生み出している。

**重要なポイントの強調:**

* **「整理」と「整頓」は異なる行為**であり、それぞれ情報理論のエントロピーと熱力学のエントロピーに対応する。
* **曖昧な指示は情報エントロピーを増大**させ、受け手の「意思決定の麻痺」を引き起こす。明確な指示は相手の情報エントロピーを下げる「整理」行為そのもの。
* 日常生活も仕事も、2種類のエントロピーとの継続的な闘いであり、**両方を意識的に下げていくことが本質的に重要**。

## 概要

Cursor Browser向けのビジュアルエディタがリリースされた。このエディタは、Webアプリ、コードベース、強力なビジュアル編集ツールを同一ウィンドウ内に統合する。要素のドラッグ操作、コンポーネントとpropsの直接インスペクション、ポイント＆クリックによる変更記述が可能となり、**デザインとコードのギャップを解消**することを目指している。

---

## 主要機能

### 1. ドラッグ＆ドロップによるレイアウト操作

- レンダリングされた要素をDOMツリー全体でドラッグ＆ドロップして直接操作可能
- **ビジュアルデザインとコーディングの統合**を実現
- ボタンの順序入れ替え、セクションの回転、グリッド構成のテストがコンテキストスイッチなしで可能
- 希望のビジュアルデザインになったら、エージェントに適用を指示
- **エージェントが関連コンポーネントを特定し、基盤となるコードを自動更新**

### 2. コンポーネント状態の直接テスト

- Reactコンポーネントのpropsをサイドバーで直接表示・編集
- **コンポーネントの様々なバリアント**を簡単にテスト可能
- propsの変更がリアルタイムでプレビューに反映

### 3. ビジュアルコントロールによるプロパティ調整

サイドバーで以下のスタイル調整が可能：

| 機能 | 詳細 |
|------|------|
| **スライダー** | 数値プロパティの微調整 |
| **カラーパレット** | ライブカラーピッカーによるプレビュー |
| **デザイントークン** | 独自のカラートークンとデザインシステムの使用 |
| **レイアウトコントロール** | グリッド、Flexboxレイアウトの調整 |
| **タイポグラフィ** | 精密なタイポグラフィ設定 |

### 4. ポイント＆プロンプト

- インターフェース上の任意の要素をクリックして変更内容を記述
- 例：
  - 要素Aをクリック → 「これを大きくして」
  - 要素Bをクリック → 「これを赤にして」
  - 要素Cをクリック → 「順序を入れ替えて」
- **複数のエージェントが並列実行**され、数秒で変更がライブ反映

## ビジョン：抽象化階層の向上

Cursorの新しいビジュアルエディタは、デザインとコード間の作業を統合し、**ユーザーが望むことをより的確に表現できるよう支援**する。実行が機構によって制限されないようにすることが目標。

### 将来の展望

- エージェントがWebアプリ構築により深く連携
- 人間が**思考からコードへより直接的につながるインターフェース**を通じてアイデアを表現
- 本機能はその方向への一歩

## まとめ

| 特徴 | 内容 |
|------|------|
| **統合性** | Webアプリ・コードベース・編集ツールを同一ウィンドウで |
| **直感性** | ドラッグ＆ドロップ、ポイント＆クリックによる操作 |
| **リアルタイム性** | 変更が即座にプレビューに反映 |
| **AI連携** | エージェントが自動でコードを更新 |
| **並列処理** | 複数の変更指示を同時に処理 |

このビジュアルエディタにより、デザイナーとエンジニアの協業がよりシームレスになり、プロダクト開発の速度と品質の向上が期待される。

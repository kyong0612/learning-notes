## 「考えない訓練」の科学──ネガティブ思考を弱める新しい方法

世の中には､ネガティブな思考に対して２つのアプローチがあるわけです。

1. 「嫌なことは、考えないようにしましょう！」みたいなアドバイス
2. 「無理に考えないようにすると逆効果だから受け入れようぜ！」みたいなアドバイス

で､近年優勢なのは後者の「ネガティブ受け入れ派」でして､この考え方は「白くま効果」と呼ばれる現象をベースにしております。これは､ある考えを思い浮かべないようにすると（たとえば「白くまを思い浮かべないで！」みたいな）、かえってそれが頭から離れなくなる、というものでして､複数の研究で存在が確認されている超有名な現象であります。

この話をもとに、心理療法の世界では「ネガティブな思考は無理に抑えないほうがいい」って考え方が生まれ、実際に治療の効果も高かったことから､ある意味で定説になりつつあったんですよ。が、最近になって、この見方をひっくり返した（かもしれない）研究([R](https://www.science.org/doi/10.1126/sciadv.adh5292))が登場したから､さぁ大変ってのが今回のお話です。

## 思考の抑圧で、むしろメンタルが改善する？

これは ケンブリッジ大学のチームが行った実験 で、要点をざっくりまとめると、こんな感じです。

- 世界16カ国から120人が参加（全員が何らかの不安・悩みを抱えていたとのこと）
- それぞれの「将来に対する不安（例：家族が病気になる、仕事を失う）」を20個書き出してもらう
- その一部について「思い出す訓練」、別の一部は「思い出さない訓練（＝思考抑制）」を実施し､3日間かけて繰り返し訓練する
- その後、抑制した思考がどう変化したか、メンタルの状態がどうなったかを検証し､3ヶ月後にもフォローアップ
  
すると、結果は従来の定説を裏切るものでして､

- ネガティブな思考を抑える訓練をした人ほど、うつ・不安・ネガティブ感情が改善し、主観的な幸福度も上がった！

って感じだったんですよ。面白いですねぇ。

この研究で使われたのは、「Imagine/No-Imagineタスク」と呼ばれるもので、ネガティブな思考（たとえば「病気になるかも！」など）に対して、 「意図的に思い出す」または 「意図的に思い出さない」って行動をトレーニングする方法です。さらに詳しくは後述しますが､こうやって「思考のオン／オフ」を自分で切り替える練習をすることで、 嫌な思考が勝手に浮かんでくる癖を弱めていこう！って考え方なんですね。

で､この方法で3日間にわたって訓練を行った参加者に､どんな変化が表れたかと言いますと､

- 抑制した思考は、記憶の鮮明さが下がった
- 不安やネガティブ感情の強さも減少
- メンタルヘルス全般に改善傾向
- しかも､この効果は高い不安・うつ・PTSD症状がある人ほど強く現れた

ということで、実に良い結果だったわけです。不思議なもんですなぁ。

## おい！「リバウンド効果」はどうなった

となると、「思考を抑えると、かえって余計に考えてしまう」という **リバウンド効果** との食い違いはどう考えればいいんだ！ってところが気になりますけども､この研究ではその点もチェックしてまして、

- 抑制した思考の記憶は、直後だけでなく3ヶ月後にも「思い出しにくい」ままだった
- 感情的なリアクション（嫌な気分など）も持続的に減っていた
- 一部で鮮明さが増した人もいたが、それは「ベースライン」の思考（＝抑圧していないもの）でも同程度に起きていた

といった報告をしております。つまり、この研究においては、リバウンド効果はほぼ確認されず、むしろ思考を抑えることでネガティブ感情が減ったわけです。

こうなると、「あれ？ネガティブ思考は抑えずに、受け入れましょう！」って考え方は間違いなの？と思う人もおりましょうが、個人的には「実は両者の考え方は根っこでつながっているのでは？」って解釈になりました。というのも、両方のアプローチは、 「 **ネガティブな思考に自動的に飲み込まれる状態から抜け出す** 」ってとこでつながっているんで。

たとえば、毎度おなじみACT（アクセプタンス＆コミットメント・セラピー）では、「ネガティブな思考をコントロールするな！」とは言うわけですが、実際にやっているのは「思考＝現実」という錯覚から距離を取る」練習なわけです。一方で、今回の研究で使われた「思考抑制」は、「思考にアクセスするルートを遮断する」ことで、結果的にその思考が持つ鮮明さや感情の力を弱めていくのを目的にしてるんですよ。つまり、どちらもやっていることは、「 **思考の力を弱めて、自分の行動を選べるようにする** 」ってことでして、その点で両者は一致しているんじゃないかと。

ただし、大きく異なるのが道具立てでして、

- **ACT** ＝「思考をそのままにしておく」ことで支配力を薄めようとする。
- **思考抑制** ＝「思考の入り口を閉じる」ことで存在感を消そうとする。

ってのがありますね。さらに明確にするために、両者の共通点を表でまとめときましょう。

| 共通点 | ポイント |
| --- | --- |
| 思考そのものに「絶対的な意味」はない！って考え方 | どちらも「思考＝現実ではない」という立場に立っているところは同じ。ただし、ACTでは「思考はただの言葉」、思考抑制では「思考は制御できる対象」と考えている。 |
| 認知の柔軟性を高めるのが目的！ | ACTは「脱フュージョン」、思考抑制は「感情や記憶の自動反応」を下げることを通じて、どちらも行動の自由度を上げる方向を目指す。 |
| トレーニングによって効果が高まる | どちらも継続的な実践によって効果が出てくる。「一発逆転」ではなく、習慣化が重要。 |
| どっちも自己に関するネガティブ思考を対象としている | ACTでは「私はダメだ」などの思考を、思考抑制でも「将来が不安」といった個人的に意味のあるイメージを対象にしている。 |
| 不安・うつ・PTSDへの効果がある | どちらのアプローチも、臨床的に効果が確認されており、とくに症状が重い人において効果が大きい傾向がある。 |

ということで、どちらも最終的には「思考に振り回されず、行動に集中する力」を高めることがなんで、「抑えるのはダメ！受け入れが正解！」と決めつけるんじゃなくて、「どちらも、ちゃんとトレーニングすれば効果がある。相性や状況で使い分ければいいじゃない」ってとこに落ち着くんじゃないでしょうか。道は違えど、ゴールは同じって感じなんで。

もちろん、この研究には注意点もありまして、

- 各国の参加者が少数すぎて、文化的バイアスやデータのばらつきが大きい
- オンライン実験のため、被験者が本当に「想像していない」かは確認できない
- 感情スケールが不連続で、変化の過大評価の可能性がある

ってあたりは「どこまで信じていいのかなー」と心配になっちゃうところです。この点はまだ判断しづらいんだけど、個人的に今回の研究は「抑圧がダメなのは、やり方が下手だからでは？」って視点を提示してくれたところがナイスだと思ってまして、つまり「正しい方法で繰り返し訓練すれば、ネガティブ思考も、安全に抑えることができるのでは？」って可能性を示したのは、やはり見るべきポイントだと思うんですよねぇ。

## ということで、「考えない訓練」も選択肢に入れていいかも

あらためてこの研究から得られる実用的なポイントをまとめると、

- ネガティブな思考を「抑える訓練」には、メンタル改善の効果がある可能性ある
- PTSDや不安が強い人ほど、効果が大きい傾向。ただし、誰にでも有効とは限らず、文化や個人差も影響する
- ACTやマインドフルネスのような「受け入れるアプローチ」との併用が理想かも

という感じです。「嫌なことは、考えちゃダメ」と言われて「そんなこと言われても無理！」と思っていた方には、ちょっと希望の持てる話だったのではないでしょうか。個人的にも、「受け入れる」だけじゃなく、「距離をとるスキル」もアリだなぁと感じさせられた次第です。

では、「考えない訓練」を実践するにはどうすればいいのか。この研究では思考抑制の詳しい手順が書かれている分けではないんだけど、参照されている文献を見る限りは、おそらく以下のような手順が使われたのではないかと。

## 思考抑制のやり方

### ステップ1：不安や心配ごとをリストアップする

まず、自分が「よく考えてしまう嫌な未来のこと」を書き出しましょう。たとえば、こんな感じです。

| キュー（きっかけ） | 内容（イメージ） |
| --- | --- |
| 検査 | 病院で深刻な病気だと診断される |
| 上司 | 職場でミスをして怒られる |
| お金 | 貯金が尽きて生活できなくなる |
| 地震 | 夜中に大地震が来て逃げ遅れる |
| 友達 | 自分だけ誘われずに疎外感を感じる |

ごらんのとおり、キューはその出来事を連想させるキーワードのことでして、このキーワードが後で思考抑制の「スイッチ」になります。

### ステップ2：2つのグループに分ける

ステップ１で書き出した不安を、以下のように分類してください。

- **Imagine（想像する）グループ** → キューが出たら、 **あえてイメージを浮かべる** 。たとえば、「上司」という文字を見た瞬間に、実際に怒られている場面を思い出して、相手の声や表情までリアルに再現するみたいな感じ。
- **No-Imagine（抑える）グループ** → キューが出たら、 **イメージを止める** 。たとえば、「検査」という文字を見て病院の映像が頭に浮かんだ瞬間に、それ以上の展開をシャットアウトして、意識を止めるようにするみたいな感じ。

それぞれ5〜10個くらいあると練習しやすいので、まずは自分にとってそこまで不安レベルが高くないものを選びましょう（あくまで練習なんで）。

### ステップ3：キューを見ながら練習開始

準備ができたら、いよいよ練習開始です。以下のように取り組んでみてください。

#### ▼ Imagine（想像）する練習

1. 特定の（例：「上司」）を見たら、
2. その出来事を、 **できるだけリアルにイメージ** する（例：会議室で上司に注意されている自分を思い浮かべ、声のトーンや表情、周囲の空気感まで感じるようにする）
3. 毎回少し違うディテールを加える（新しい展開をつけてもOK。例：前回は言い返したけど、今回は黙って聞いている自分を想像する）

#### ▼ No-Imagine（抑える）練習

1. キュー（例：「検査」）を見たら、
2. 一瞬だけ「病気のイメージ」が浮かんでもOKなので、
3. すぐに **思考の進行を止めようとしてみる。この時には、何も考えず、頭を空っぽにすることをめざす。**
4. 「何か別のことを考える」のではなく、できるだけ **無の状態を目指す**

このステップのポイントは、「置き換え」ではなく「遮断」なので、他のポジティブなことに逃げないようにしましょう。たとえば「病気のことを考えない代わりに楽しいことを考えよう」はNGだけど、「あ、今イメージが浮かんだな」と気づいて、ただスッと意識を止める」ならOKであります。

### ステップ4：1日10分〜15分、数セット繰り返す

ステップ３は、だいたい以下のようなペースで練習してください。

- ImagineとNo-Imagineを交互に、キュー10個ずつ（そんなにない時は５個でもOK）
- 各キューに対して4〜5秒間、集中して練習
- これを **1日1〜2セット × 3日間くらい** 継続

この繰り返しで、少しずつ思考のコントロール力が向上していくと思われますんで。

ここでの実践のコツとしては、まずは完全に「何も浮かばないようにしよう」と力むと逆効果になるんでご注意ください。「浮かんでもいいけど、止める」くらいのゆるいスタンスでいったほうが上手くいくはずであります。もし実践が難しければ、目を開けて実際のキューを見つめながら無思考に集中するのもアリ。想像が暴走しそうなら、 **深呼吸しながら5秒だけ目を閉じてみる** のも有効とされております。

## このトレーニングを継続すると、

- 嫌なことを **考え続けてしまうクセ** が減る
- 思考に **主導権を取り戻せる感覚** が出てくる
- 結果的に、不安や落ち込みの感情が減る

といった効果が見られる可能性がありますんで、「思考の受容」がうまくいかない方は、こちらも試してみてくださいませ。

## 「思考の抑制」と「思考の受容」を使い分けようぜ

ってことで、おおかたの効果がわかったところで、最後に「思考の抑制」と「思考の受容（ACTやマインドフルネス）」を **どう使い分ければいいのか？** ってところを考えてみましょう。このポイントは今回の文献で触れられてはいないので、あくまで私の個人的な見解だとお考えください。

まず、思考抑制が有効そうな場面としては、

- **思考の抑制が効きやすそうなタイミング**
  - 頭にこびりつくネガティブイメージがあるとき
  - 不安やストレスで集中力が低下しているとき
  - トラウマ的な記憶がよくフラッシュバックするとき
  - 思考のループから物理的に抜け出したいとき

みたいになるでしょう。このアプローチは、 **意識的に思考を遮断するスキル** を使うので「感情や記憶の侵入」が弱まりやすく、短期的に自分のコントロール感を取り戻すことができるんじゃないかと。たとえば、プレゼン直前や寝る前など、「今はこの思考に巻き込まれてる場合じゃない」という場面には特に有効でしょうな。

一方で、思考受容が有効そうな場面としては、

- **思考の受容が効きやすそうなタイミング**
  - いつも自動的に「自分はダメだ」「嫌われている」と考えてしまうとき
  - ネガティブ思考をなくすことに疲れてしまったとき
  - 自分の感情や思考との「関係の仕方」を変えたいとき
  - 感情に巻き込まれながらも、自分のやりたいことに向かいたいとき

みたいになるでしょう。このアプローチは、「思考そのものを変えるのではなく、その受け止め方や反応の仕方を変える」ことで、長期的な柔軟性とメンタルの安定性を育ててくれますんで。総じて「慢性的なネガティブ思考」や「価値観ベースの行動」に向いているんじゃないでしょうか。

なので、それぞれが向いている人の特徴もまとめておくと、以下のようになるかと思うわけです。

- **思考の抑制が向いている人**
  - 頭の中でイメージがリアルに浮かぶタイプ（想像力が強い）
  - 一時的にでも感情の波を弱めたい人
  - 実験的なアプローチや「訓練」に対して抵抗がない人
  - 抑制を「コントロール手段」ではなく、「筋トレ」的にとらえられる人
  - ACTやマインドフルネスで逆に混乱した経験がある人

思考の抑制については、「嫌な思考を抑えたい」という欲求が強くても、それを力づくでねじ伏せようとせず、淡々と訓練する姿勢が持てるかどうかがポイントなんで、上記の要素を満たす人には適してるんじゃないでしょうか。

一方で、思考受容が向いてそうな場面としては、

- **思考の受容が向いている人**
  - 感情を抑えようとするほど、逆に強く感じてしまうタイプ
  - 完璧主義や「こうあるべき思考」が強い人
  - 抑えようとすると自己批判が止まらなくなる傾向がある人
  - 瞑想やジャーナリングなど「観察系」の手法が好きな人
  - 長期的に「思考に振り回されない生き方」を身につけたい人

といったあたりになるでしょうね。こちらは、「ネガティブな思考や感情を排除しようとしないことで、逆に自由になれる」という **逆説的な考え方** にしっくりくるかどうかがカギになるかと。

ただし、この問題については「どちらが正解で不正解！」ってわけじゃないので、私としては場面によって使い分けることをお勧めします。たとえば、

- 「寝る前に将来の不安がぐるぐる回り始めた」→ 抑制タスクで一時的に遮断する
- 「仕事中にふと浮かんだ自己否定」→ 受容スキルでそのまま観察して流す
- 「過去のトラウマが何度も鮮明に蘇ってくる」→ 抑制トレーニングで鮮明さを下げる
- 「自分の感情に振り回されて行動が止まる」→ ACT的に価値観へ意識を戻す

みたいな感じっすね。このように、「 **一方を否定するのではなく、目的に応じて使い分ける** 」ことが、最終的には最も強力なアプローチになるんじゃないかと。

いずれにせよ、思考の抑制も「思考に巻き込まれず、自分の行動に戻るための方法」として捉えれば、有効に使える可能性は大。「思考アクセスの遮断」によって自由になろうとするのであれば、どちらも有効に使えるんじゃないでしょうか。とにかく **今の自分にとってしっくりくる方から試して** みてくださいませ。

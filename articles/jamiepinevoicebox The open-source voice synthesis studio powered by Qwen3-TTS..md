## Voicebox とは

Voiceboxは、**ローカルファーストの音声クローニングスタジオ**で、プロフェッショナルな音声合成のためのDAW（デジタルオーディオワークステーション）ライクな機能を備えている。**ElevenLabsのローカル・無料・オープンソース代替**として位置づけられており、モデルのダウンロード、音声クローン、音声生成をすべて自分のマシン上で完結させることができる。

### 主要な差別化ポイント

- **完全なプライバシー** — モデルと音声データはすべてローカルに保存
- **プロフェッショナルツール** — マルチトラックタイムラインエディタ、オーディオトリミング、会話ミキシング
- **モデルの柔軟性** — 現在Qwen3-TTSを搭載、XTTS・Barkなど他モデルのサポートも予定
- **APIファースト** — デスクトップアプリとしても、REST API経由で自分のプロジェクトに統合しても使用可能
- **ネイティブパフォーマンス** — Tauri（Rust）で構築（Electronではない）
- **Mac上での高速推論** — Apple Silicon上でMLXバックエンドによるMetal加速で4〜5倍高速

Pythonのインストール不要、クラウド依存なし、使用制限なし。

## 主要機能

### 1. Qwen3-TTSによる音声クローニング

Alibabaの**Qwen3-TTS**を搭載。わずか数秒のオーディオサンプルからほぼ完璧な音声クローンを実現する。

- **即時クローニング** — サンプルをアップロードするだけでボイスプロファイルを生成
- **高忠実度** — 自然な韻律、感情、リズムを再現
- **多言語対応** — 英語、中国語、その他追加予定
- **Mac上で超高速** — MLXバックエンドがApple SiliconのNeural Engineを活用

### 2. ボイスプロファイル管理

- オーディオファイルからプロファイルを作成、またはアプリ内で直接録音
- プロファイルのインポート/エクスポートに対応
- **マルチサンプルサポート** — 複数のサンプルを組み合わせてクローン品質を向上
- 説明や言語タグで整理

### 3. 音声生成（Text-to-Speech）

- クローンした任意の音声でテキスト読み上げ
- 長文コンテンツのバッチ生成に対応
- **スマートキャッシュ** — ボイスプロンプトキャッシュによる即時再生成

### 4. Stories Editor（マルチボイスナラティブ）

タイムラインベースのエディタで、マルチボイスのナラティブ・ポッドキャスト・会話を作成。

- **マルチトラック構成** — 複数の音声トラックを1つのプロジェクトに配置
- **インラインオーディオ編集** — タイムライン上でクリップを直接トリミング・分割
- **自動再生** — 同期プレイヘッドによるストーリーのプレビュー
- **ボイスミキシング** — 複数の参加者による会話構築

### 5. 録音と文字起こし

- 波形可視化付きのアプリ内録音
- **システムオーディオキャプチャ** — macOSとWindowsでデスクトップオーディオを録音
- **Whisper**による自動文字起こし
- 複数フォーマットでのエクスポート

### 6. 生成履歴

- すべての生成オーディオの完全な履歴
- 音声、テキスト、日付による検索・フィルター
- ワンクリックで過去の生成を再実行

## REST API

Voiceboxは完全なREST APIを公開しており、自分のアプリに音声合成を統合可能。

```shell
# 音声生成
curl -X POST http://localhost:8000/generate \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello world", "profile_id": "abc123", "language": "en"}'

# ボイスプロファイル一覧
curl http://localhost:8000/profiles
```

**主なユースケース**: ゲーム対話システム、ポッドキャスト/動画制作、アクセシビリティツール、音声アシスタント、コンテンツ制作自動化

## 技術スタック

| レイヤー | 技術 |
|---|---|
| デスクトップアプリ | Tauri (Rust) |
| フロントエンド | React, TypeScript, Tailwind CSS |
| 状態管理 | Zustand, React Query |
| バックエンド | FastAPI (Python) |
| 音声モデル | Qwen3-TTS (PyTorch or MLX) |
| 文字起こし | Whisper (PyTorch or MLX) |
| 推論エンジン | MLX (Apple Silicon) / PyTorch (Windows/Linux/Intel) |
| データベース | SQLite |
| オーディオ | WaveSurfer.js, librosa |

**Tauriを選択した理由**: Electronと比較して10倍小さいバンドル、ネイティブパフォーマンス、低メモリ消費。

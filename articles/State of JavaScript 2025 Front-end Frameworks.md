## 要約

State of JavaScript 2025調査のフロントエンドフレームワーク部門の結果。フロントエンドエコシステムは「常に混乱している」という印象に反し、この1年間は**驚くほど安定**していることが明らかになった。

---

## Library Highlight: Solid

- **Solid**は回答者の10%にしか使われていないが、**5年連続で満足度1位**を維持
- 作者の[Ryan Carniato](https://dev.to/ryansolid)は**Signals**の概念を広めた立役者
- 他のフロントエンドフレームワークの仕組みを解説する能力にも長けている

## フロントエンドフレームワーク 経年比較（Ratios Over Time）

### 使用率（Usage）
- 過去1年間で使用率ランキングに**ほぼ変動なし**
- 唯一の変化: **Alpine.js**と**HTMX**の順位が入れ替わった

### 満足度（Satisfaction）
- こちらも安定しているが、**Qwik**は引き続きランクを下げている

### 関心度（Interest）
- 新たなハイプの欠如が顕著
- **Solid**と**Preact**のみが昨年から関心度を伸ばした

対象フレームワーク: React, Vue.js, Angular, Preact, Svelte, Alpine.js, Lit, Solid, Qwik, Stencil, HTMX

## 主要な発見・結論

1. **エコシステムの安定化**: フロントエンドフレームワークのランキングは驚くほど安定しており、「常に混乱している」というイメージとは異なる現実が浮き彫りになった
2. **Solidの持続的な高評価**: 使用率は低いものの、5年連続満足度1位はその技術的優位性を示す
3. **フレームワーク乗り換えの神話**: 平均2.6個しか使っておらず、頻繁な乗り換えは実態と異なる
4. **React の二面性**: 最も使われるフレームワークであると同時に、最大のペインポイントでもある
5. **新たなハイプの不在**: 関心度で上昇しているのはSolidとPreactのみで、新しいフレームワークへの熱狂は落ち着いている
6. **Ripple**: 「その他」カテゴリで注目される新規参入フレームワーク

## 要旨

株式会社マネーフォワードは2026年3月26日、クラウド型会計ソフト『マネーフォワード クラウド会計』において、**リモートMCP（Model Context Protocol）サーバー**と**外部連携API**を全プランのユーザーへ公開した。これにより、AIエージェントを通じた仕訳入力・帳簿検索・レポート作成などの会計業務の自動化が、すべてのユーザーに開放された。

## MCPサーバーの提供

### MCPサーバーとは

MCPサーバーは、**AIエージェント（MCPクライアント）と『マネーフォワード クラウド会計』を接続する仕組み**。AIエージェントがユーザーに代わって会計ソフト上の操作を自律的に代行する。

### AIエージェントで実行可能な会計業務

- 仕訳入力
- 帳簿検索
- データ確認
- レポート作成

### 対応AIツール

| ツール名 | 種別 |
|---|---|
| Claude Desktop | AIアシスタント |
| Claude Code | 開発者向けAI |
| Claude Cowork | チーム向けAI |
| Cursor | AI搭載IDE |
| Gemini CLI | Google AI CLI |

### ローカル vs リモートMCPサーバー

| | ローカルMCP | リモートMCP |
|---|---|---|
| 動作場所 | ユーザーのPC上 | マネーフォワードのクラウドサーバー上 |
| 環境構築 | ユーザー自身で実行環境の構築・管理が必要 | **不要**（接続設定のみ） |
| セットアップ | 複雑 | **簡単** |
| 特徴 | 手元でダイレクトにアクセス | セキュアかつスピーディーに利用開始 |

マネーフォワードはβ版の開始当初よりリモートMCPサーバーを提供しており、今回それを全プランに開放した。

### 今後のロードマップ

- MCPサーバー経由での**認証時間の延長**
- **再認証の自動化**

## 提供背景

マネーフォワードは「**クラウドからAIへ**」というビジョンのもと、SaaSとAIが融合した新たなUXの実現を掲げている。

### これまでの提供範囲（限定公開時）

| サービス | 対象 |
|---|---|
| MCPサーバー（β版） | プラチナランク以上の公認メンバーの士業事務所 |
| API | シルバーランク以上の公認メンバーの士業事務所 |

AIエージェントが自律的に業務を遂行する環境への期待が高まる中、より多くのユーザーにデータ活用とAI自動化の価値を届けるため、**全プランへの公開を決定**した。

## 先行利用ユーザーの声

**セブンリッチ会計事務所**（代表取締役社長・所長 服部峻介氏、税理士 外林洋輝氏）:

- 『マネーフォワード クラウド会計』の導入で会計業務を効率化したが、「人が操作すること」が前提だった
- MCPサーバー公開により、入力からレビューまでの一連の会計業務を**人を介さず自動化**できる可能性を感じた
- 現在は**Claudeと複数のSkillsを組み合わせ**、一部の顧問先で自動化を実現
- 今後は全顧問先の会計業務自動化を目指し、マネーフォワードと共にMCPを進化させる方針

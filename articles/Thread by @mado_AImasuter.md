## 要約

### 核心メッセージ：まじめに申告している人ほど損をしている

元国税局職員（36年間勤務）のマド氏が、調査の最前線で得た知見を共有。**真面目に申告している人ほど、知識不足により大きく損をしている**という国税現場の真実を明かしている。

### 具体例：個人事業主の税負担

年商2,000万円・利益800万円の個人事業主の場合：

| 税目 | 金額 |
|------|------|
| 所得税 | 53万円 |
| 住民税 | 48万円 |
| 事業税 | 24万円 |
| 消費税（2割特例） | 16万円 |
| **合計** | **127万円** |

これは「普通の申告」をした場合の数字であり、知識の差によってこの金額は大きく変わりうる。

### 節税の誤解と真実

**❌ 誤解**：節税＝経費を増やすこと

**✅ 真実**：本当に効く節税は**制度の選択と設計**で決まる

現場で否認される申告のほとんどが、この間違った思い込みから始まっている。

### 否認されにくい節税手法

国税側から見ても正しく機能していた節税：

1. **小規模企業共済** - 全額所得控除
2. **倒産防止共済の掛金戦略** - 掛金の柔軟な運用
3. **専従者給与による所得分散** - 精密な設計が必要
4. **消費税2割特例の選択タイミング** - 見極めが重要
5. **社会保険への移行** - 法人成りも視野に

### 税理士任せの危険性

> 「税理士に任せていれば安心」という思い込みは危険

- 税理士が悪いわけではない
- しかし、その申告があなたにとって**最適かどうかは保証されない**
- **納税者本人が仕組みを理解していない申告ほど脆い**

### 重要な結論

- **知識の差だけで税負担が何十万〜百万円単位で変わる**
- 申告直前ではできることがほとんどない
- **今のタイミング（事前準備の段階）が決定的に重要**

### 著者の経歴

- 国税局で36年間勤務
- 特別調査、国際税務を経験
- 調査部で27年間勤務
- 東京局・大阪局両方で勤務経験あり

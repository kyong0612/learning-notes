## 概要

Xで半年間毎日投稿してフォロワー50人だった著者（寝稼ぎ）が、Threadsに切り替えて1年7ヶ月で累計1億の売上を達成。スレッズ攻略コミュニティを運営し、数百アカウントのデータを日々分析する中で見出した、XとThreadsのアルゴリズムの構造的違いと、Threadsで成果を出すための実践的戦略をまとめた記事。

---

## 1. アルゴリズムの核心的違い：「人」vs「投稿」

XとThreadsのアルゴリズムの最大の違いは **「誰に届けるか」の設計思想** にある。

| | X | Threads |
|---|---|---|
| **配信ロジック** | 「この**人**の投稿をまた見せよう」 | 「この**投稿**に合う人を探そう」 |
| **仕組み** | 過去に反応した人に次の投稿も優先配信 | 毎回ゼロから最適な受け手を探す |
| **フォロワーの意味** | 「資産」として機能（繰り返し届く） | **ほぼ無意味**（1万人いても次の投稿を見るのはほぼ0） |

> **重要な帰結**: Threadsではフォロワー数の価値がほぼなく、フォロワー1万・売上月4万のような「ハリボテアカウント」が大量発生している。追うべきは「売上」にコミットした運用。

---

## 2. ジャンルの壁の有無

| | X | Threads |
|---|---|---|
| **ジャンル分け** | 非常に強い（いいね履歴でTLがジャンル一色に） | **ジャンルの壁がほぼない** |
| **類似アカウント推薦** | クラスタリングが強力 | 異ジャンルがTLに混在して表示 |
| **新規参入のしやすさ** | 既存の壁に阻まれやすい | 後発・フォロワー0でも数万人にリーチ可能 |

**Threadsでジャンルの壁がない理由**: ユーザーの大半が30〜50代女性で、偶然の発見（セレンディピティ）を楽しむ傾向がある。Meta社も「人々が新しいコミュニティやクリエイターと出会うことを重視する」と公式に表明。

**実績データ**:
- 開始初日にフォロワー0で投稿が**5万インプレッション**超え
- たった1投稿で**129万インプレッション**、その1投稿だけで**4,000フォロワー**増加

## 3. 初動評価の速度：「5分〜1時間」が勝負

| | X | Threads |
|---|---|---|
| **評価速度** | 数時間〜数日かけてじわじわ伸びる | **投稿後5分〜1時間でほぼ勝負が決まる** |
| **投稿タイミング** | そこまで神経質にならなくてよい | **極めて重要** |

**Threadsの初動メカニズム**:
1. 初動1時間以内に **1,000〜2,000インプレッション** → アルゴリズムが「反応がいい」と判断 → 次の層に配信
2. さらに反応が良ければ次の層へ → 数万〜数十万まで連鎖的に伸びる
3. 初動1時間で100インプレッションしか取れなければ → **配信ストップ**

**推奨投稿時間帯**: **朝6時台・昼12時台・夜21時台**（アクティブ率集中タイミング）

> 同じ内容でも時間帯を変えるだけでインプレッションが **5倍以上** 変わることがある。

## 4. 収益化速度の比較

「ゼロから初収益」までの平均期間（著者の実体験と周囲のデータ）:

| SNS | 初収益までの期間 |
|---|---|
| YouTube | 6ヶ月〜1年 |
| Instagram | 4〜6ヶ月 |
| X | 2〜3ヶ月 |
| **Threads** | **最短2日、平均1〜2ヶ月** |

**Threadsの収益化が速い理由**: Meta社がInstagramを15年以上運営して蓄積した膨大なユーザー属性データ（年齢・興味関心・行動パターン等）をThreadsの投稿配信にそのまま転用。MetaのAIが「最も反応しそうな人」を自動で選んで無料で届けてくれる。

**実際の指導事例**:
- **A**: Xで半年間鳴かず飛ばず → Threads切替後2ヶ月で累計**1,300万円**（フォロワー4,000人）
- **B**: ダイエット系でnote460部売切り、セミナー同時接続220人（フォロワー4,000人）
- **C**: 50代・SNS完全未経験 → 初月で**月3万**達成（1日1時間の作業）
- **D**: スピ系完全初心者 → 初月で**月100万超え**

## 5. Threadsの致命的弱点：フォロワーの質が薄い

**構造的問題**: 毎回違う人に投稿が届くため、マーケティングの「単純接触効果」（繰り返し接触によるファン化）が構造的に起きにくい。

**XとThreadsのファン化の違い**:
- **X**: いいね → 次も届く → 何度も目にする → 覚える → 共感 → フォロー → ファン化 (**自動で回る**)
- **Threads**: バズっても翌日その人たちには届かない → **ファン化が起きない**

**典型的な失敗パターン**:
| 時期 | 状態 |
|---|---|
| 1〜2ヶ月目 | リーチの爆発力で新規流入 → 売れる |
| 3ヶ月目 | 新規は来るがファン化されず → 離脱増加 |
| 4ヶ月目 | マインドシェア低下 → リピートなし → **売上激減** |

> 月100万超えていたのに半年後にゼロ、というケースを **5件以上** 確認。

**追うべきKPI**: フォロワー数やインプレッションではなく、**「リスト流入数」と「リスト内の反応率」**。

## 6. 「3ヶ月で売上消滅」を防ぐ導線設計

核心は **「集客・教育・販売を全部別の場所でやる」** こと。Threads単体で完結しようとすると構造的に詰む。

### 導線設計テンプレート

```
① Threads投稿 → 新規リーチ（売らない。「面白い」「もっと知りたい」と思わせるだけ）
    ↓
② オープンチャット（オプチャ）に誘導 → 流れない場所でのコミュニケーション
    ↓
③ オプチャで教育コンテンツ配信 → 単純接触効果を人工的に作り出す
    ↓
④ 公式LINEで販売 → 信頼が積み上がった状態で商品案内 → 売り込まずに売れる
```

### オプチャ vs 公式LINE 開封率比較（著者実測データ）

| チャネル | 開封率 |
|---|---|
| 公式LINE | 1,174 / 2,271 = **51%** |
| オープンチャット | 1,995 / 2,230 = **89%** |

> 失速する人は **例外なくこの導線を持っていなかった**。月100万超えから半年後にゼロになった人は全員Threads投稿だけで完結しようとしていた。

## 7. リンクペナルティの罠

Threadsでは **リンク付き投稿のインプレッションが意図的に下げられる**。Meta社はユーザーをアプリ内に留めたいため、外部リンクにペナルティを課す。

### リンクペナルティ対策

| 方法 | 評価 | 説明 |
|---|---|---|
| 本文にリンクを直接貼る | ✕ | インプレッション激減 |
| コメント欄にリンクを貼る | △ | 若干ペナルティあり |
| **「プロフのリンクから」で誘導** | **◎** | 王道・安全 |

> Xではリンク付き投稿にここまで明確なペナルティはないため、Xと同じ感覚で運用すると永遠に伸びない。

## 8. XからThreadsに切り替える人の5つの致命的ミス

| # | ミス | 問題点 |
|---|---|---|
| ① | **Xと同じ投稿をそのまま貼る** | Threadsの読者は毎回初見。投稿単体で完結する内容でないと離脱される |
| ② | **フォロワー数を追いかける** | アルゴリズムがフォロワーに優先配信しないため意味がない。追うべきは「投稿ごとのインプレッション」と「リスト流入数」 |
| ③ | **有益情報だけを出し続ける** | インプレッションは伸びるが売上には繋がらない。売上を作る人は**欲求喚起3：有益情報1**の比率で運用 |
| ④ | **投稿だけで完結しようとする** | Threads投稿は「集客」であり「販売」ではない。別の場所での教育・販売導線が必須 |
| ⑤ | **投稿タイミングを考えない** | 初動5分〜1時間が全て。朝6時に出すだけでインプレッションが5倍になることも |

## 9. XとThreadsの特徴 完全比較

### X（旧Twitter）
- フォロワーに繰り返し届く → **ファン化が自然に起きる** → X完結で大きくマネタイズ可能
- ジャンルの壁が厚い → 新規参入が難しい
- 投稿タイミングに左右されにくい → 実力主義
- 既にポジションや関係値を持つ人が有利

### Threads
- 新規への爆発的リーチ → **フォロワー0でもチャンスあり**
- ジャンルの壁がない → 後発でもすぐ結果が出せる
- キーワードで特定属性にピンポイント配信 → **無料の広告配信**
- 初動5分〜1時間が勝負 → 投稿時間帯と構成が超重要
- ファン化は自動では起きない → **導線設計が必須**
- 収益化速度が全SNS中トップ → 最短10日で初収益

---

## 10. Threadsで結果を出す3つの必須条件

| # | 条件 | 具体的な内容 |
|---|---|---|
| ① | **導線を組む** | 集客・教育・販売を分離する（Threads → オプチャ → LINE → 販売） |
| ② | **初動を設計する** | 投稿時間帯を固定し、一文目のインパクトと元投稿7行以下を徹底 |
| ③ | **キーワードを意識する** | 届けたい人の属性ワードと固有名詞を投稿に入れる |

> **結論**: SNS運用は才能のゲームではなく「情報のゲーム」。感覚でやる人と構造を理解している人では、同じ労力でも結果に10倍の差が出る。

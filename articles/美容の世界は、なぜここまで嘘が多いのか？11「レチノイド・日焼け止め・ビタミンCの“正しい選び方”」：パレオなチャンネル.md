## 1. レチノイド

### 選び方の基本原則
- **初心者**: レチノール or レチナールからスタート
- **上級者**: レチノイン酸誘導体（HPR、トレチノイン）へ移行
- **低刺激のコツ**: 「カプセル化」「バッファリング」された製品を選ぶ

### おすすめ製品

#### The Ordinary レチノール 0.2% / 0.5% / 1% in スクワラン
| メリット | デメリット |
|---------|-----------|
| 3段階の濃度で肌の耐性に合わせて調整可能 | 空気に触れると酸化が進む |
| スクワランベースで乾燥・刺激を軽減 | オイルベースで脂性肌にはベタつく可能性 |
| シンプルな処方、添加物が少ない | 後続スキンケアとの相性に注意 |

> **保管のポイント**: 開封後の期限を守り、暗冷所で保管

#### Life-flo レチノールA 1% アドバンスド・リバイタリゼーションクリーム
| メリット | デメリット |
|---------|-----------|
| 高濃度でコラーゲン生成・シワ改善に効果的 | 赤み・皮剥け・乾燥の副作用リスク |
| 遮光性の高いジャー容器 | 初心者向けではない |
| 定番の高濃度レチノール | ビタミンC・酸との併用に注意 |

#### 資生堂 エリクシール シュペリエル エンリッチド リンクルクリーム
| メリット | デメリット |
|---------|-----------|
| 独自の安定化技術で品質が安定 | 価格が高い |
| 酸素侵入を防ぐ特殊密封チューブ | A反応（赤み・皮剥け）の可能性 |
| 保湿成分豊富で乾燥を軽減 | 日中使用時は日焼け止め必須 |

## 2. 日焼け止め

### 選び方の基本原則
- **UVA（PA値）とUVB（SPF値）両方の防御**が必要
- 「日常用」と「レジャー用」で分ける
- 敏感肌は**紫外線吸収剤不使用（ノンケミカル）**を選ぶ
- **最重要ポイント**: 推奨量を毎日心地よく塗り続けられるか

### 日常使い用

#### ビオレUV アクアリッチ ウォータリーエッセンス
| メリット | デメリット |
|---------|-----------|
| ミクロディフェンス処方で塗りムラ防止 | アルコール（エタノール）含有量が多い |
| ベタつき・白浮きなし | 極度の乾燥肌・敏感肌には刺激の可能性 |
| 水のような軽い質感 | 激しいスポーツ・海水浴には不向き |
| メイク下地としても優秀 | |

### 敏感肌用

#### キュレル UVエッセンス SPF30/PA+++
| メリット | デメリット |
|---------|-----------|
| ノンケミカルで低刺激 | 白浮きの可能性（地肌が暗めの方は注意） |
| セラミド配合で肌バリア保護 | |
| SPF30で日常生活に十分な防御力 | |

### レジャー用

#### アネッサ パーフェクトUV スキンケアミルク N SPF50+/PA++++
| メリット | デメリット |
|---------|-----------|
| オートリペア技術で汗・水・摩擦に強い | 丁寧なクレンジングが必要 |
| 広範囲の紫外線を効率よくブロック | エタノール含有（敏感肌は注意） |
| 約50%がスキンケア成分で乾燥防止 | |

### 光老化ケア重視用

#### ラロッシュポゼ UVイデア XL プロテクション トーンアップ
| メリット | デメリット |
|---------|-----------|
| UVA/UVB両方に高い安定した防御力 | テクスチャーが重め（脂性肌はテカリ注意） |
| PM2.5などの汚染物質からも保護 | トーンアップ成分で白浮きの可能性 |
| 低刺激設計・保湿成分配合 | |
| レチノール使用中の肌にも適している | |

## 3. ビタミンC美容液

### 選び方の基本原則
- **安定型（アスコルビン酸誘導体）** or **純粋型（ピュアビタミンC）**を選択
- pHと濃度が重要
- 酸化しにくいボトル（エアレス or 遮光）を使用しているか確認

### おすすめ製品

#### ナールスピュア（VCエチル 5% + セラミド）
| メリット | デメリット |
|---------|-----------|
| VCエチルは酸化しにくい安定型 | 純粋ビタミンC並みの刺激を感じる場合あり |
| 酵素変換不要で比較的早く効果発揮 | 敏感肌はピリピリ・乾燥に注意 |
| エイジングケア全般に有効 | |

#### Geek & Gorgeous C-Glow（15% L-アスコルビン酸 + フェルラ酸 + ビタミンE）
| メリット | デメリット |
|---------|-----------|
| 科学的に最も効果が証明された組み合わせ | 使用期限が約3ヶ月と短い |
| 最適なpH調整（3.2〜3.4） | 高濃度・低pHで敏感肌に刺激 |
| 毎週作りたてを発送 | 冷蔵庫保存が必要 |

#### Mad Hippie ビタミンCセラム（SAP）
| メリット | デメリット |
|---------|-----------|
| 非常に安定し変色しにくい | 即効性は純粋ビタミンCより穏やか |
| pHが肌に近く刺激が少ない | 目に見える変化に時間がかかる可能性 |
| 抗菌作用でニキビ肌にも有効 | |
| フェルラ酸・ビタミンE配合 | |

#### Obagi C25 セラム ネオ（25% L-アスコルビン酸）
| メリット | デメリット |
|---------|-----------|
| 市販品トップクラスの高濃度 | 敏感肌は痛み・乾燥を感じやすい |
| 直接作用で変換不要 | 12mLで11,000円と高価 |
| 独自技術で鮮度維持 | |
| これ1つで多くの悩みに対処可能 | |

> **コスパ重視なら**: メラノCCなど同成分の安価な誘導体製品を検討

## 4. シャンプー・トリートメント

### 基本の考え方
- **髪は死んだ細胞なので修復は不可能**
- 補修成分が髪表面に残ることが重要
- **シリコンは悪ではない**（むしろ摩擦防止に有効）
- スカルプ重視系は洗浄力が強すぎる場合あり

### おすすめ製品

#### PaleWash パレウォッシュ
| メリット | デメリット |
|---------|-----------|
| 極めて低刺激な洗浄剤 | モコモコとした泡立ちはない |
| 余分な成分を極力排除 | コンディショニング成分なしで髪がキシむ |
| かなりの敏感肌でも使用可能 | 別途トリートメントが必要 |
| アレルギー・刺激リスクがほぼない | |

#### The Ordinary サルフェート 4% クレンザー
| メリット | デメリット |
|---------|-----------|
| SLES濃度4%で刺激を最小限に | 泡立ちが少ない |
| 髪にも体にも使えるシンプル処方 | 使用後に髪がキシむ |
| 不要な成分が入っていない | |

#### The Ordinary Behentrimonium Chloride 2% Conditioner
| メリット | デメリット |
|---------|-----------|
| カチオン性成分が髪に吸着し摩擦軽減 | 激しく傷んだ髪には物足りない可能性 |
| シリコンなしで髪が重くならない | 極度のダメージ毛にはシリコン配合製品を推奨 |
| 髪が細い方・ペタンとしやすい方に最適 | |
| 不要な香料・抽出物なしで敏感な頭皮にも対応 | |

#### パンテーン ミラクルズ ヘアオイル（洗い流さないトリートメント）
| メリット | デメリット |
|---------|-----------|
| シリコン配合で髪の摩擦軽減 | 毎日洗い流さないとベタつき・重さの原因に |
| キューティクル保護 | 定期的に洗浄力の高いシャンプーでリセットが必要 |

## まとめ

### カテゴリ別の選び方チェックリスト

| カテゴリ | 初心者向け | 上級者向け |
|---------|-----------|-----------|
| レチノイド | The Ordinary 0.2% | Life-flo 1% / 資生堂エリクシール |
| 日焼け止め（日常） | ビオレUV アクアリッチ | ラロッシュポゼ UVイデア |
| 日焼け止め（敏感肌） | キュレル UVエッセンス | - |
| 日焼け止め（レジャー） | アネッサ | アネッサ |
| ビタミンC | Mad Hippie / ナールスピュア | Obagi C25 / Geek & Gorgeous |
| シャンプー | PaleWash / The Ordinary 4% | 同左 |
| コンディショナー | The Ordinary 2% | パンテーン ヘアオイル |

### 重要なポイント
1. **レチノイド**: 低濃度から始めて肌を慣らす「レチノイド・トレランス」期間が必須
2. **日焼け止め**: 毎日塗り続けられる使用感が最も重要
3. **ビタミンC**: 酸化に注意、開封後の保管期限を守る
4. **ヘアケア**: 髪の「修復」は不可能、表面保護に集中する

## chezmoi - 複数マシン間でのdotfiles管理ツール

### 概要

chezmoi（発音: /ʃeɪ mwa/、shay-mwa）は、複数の異なるマシン間で個人設定ファイル（dotfiles、例：`~/.gitconfig`）を安全に管理するためのツールです。シンボリックリンクやベアGitリポジトリを超えた多くの機能を提供します。

最新バージョンは2.67.0です。

## 主な機能

chezmoiは以下の機能を提供します：

### 1. テンプレート機能

- マシン間の小さな違いを処理するためのテンプレート機能
- 環境に応じた設定の自動調整

### 2. パスワードマネージャー連携

以下のパスワードマネージャーをサポート：

- 1Password
- AWS Secrets Manager
- Azure Key Vault
- Bitwarden
- Dashlane
- Doppler
- ejson
- gopass
- KeePassXC
- Keychain（macOS）とWindows Credentials Manager
- Keeper
- LastPass
- pass / passhole
- Vault
- カスタムパスワードマネージャー

### 3. ファイル暗号化

- GPGまたはageを使用した完全なファイル暗号化
- 機密情報の安全な保存

### 4. その他の機能

- アーカイブからのファイルインポート（シェルやエディタプラグインに最適）
- スクリプト実行（その他の処理に対応）
- 外部ファイルのインクルード

## インストールと使用方法

### インストール

chezmoiは以下の方法でインストールできます：

1. **curl経由のインストール**（推奨）:

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_USERNAME
```

1. **パッケージマネージャー経由**:

   - 主要なパッケージマネージャーに対応

### 基本的な使用方法

**新しいマシンでの初期セットアップ**:

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_USERNAME
```

**dotfilesの更新**:

```bash
chezmoi update
```

## 技術的特徴

- **クロスプラットフォーム**: すべての主要なオペレーティングシステムで動作
- **シングルバイナリ**: 依存関係のない単一の静的リンクされたバイナリとして配布
- **root権限不要**: rootアクセスを必要としない
- **GitHub統合**: GitHub dotfilesリポジトリとの統合

## まとめ

chezmoiは、複数のマシン間でdotfilesを管理するための強力で柔軟なツールです。テンプレート、パスワードマネージャー統合、暗号化などの高度な機能により、シンボリックリンクやベアGitリポジトリを超えた管理が可能になります。単一のコマンドで新しいマシンに設定を展開できるため、開発環境のセットアップが大幅に簡素化されます。

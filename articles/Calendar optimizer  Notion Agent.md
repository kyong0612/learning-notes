## 概要

**Calendar optimizer** は、Notionが提供するカスタムAIエージェントテンプレートの一つ。日々のカレンダーを自動的に分析し、スケジュール管理を最適化する。

## 主な機能

| 機能 | 説明 |
|------|------|
| **会議コンフリクト検出** | 日々のカレンダーから会議の重複・競合を自動的に特定 |
| **スケジュール過負荷の検知** | 1日あたりの会議時間が過多になっていないかを検出 |
| **優先度のリコメンド** | どの会議を優先し、どれをリスケジュールすべきかを提案 |
| **代替タイムスロット提案** | 全出席者の空き時間を考慮した代替時間をプロアクティブに通知 |

## カスタマイズ可能な設定

エージェントの指示（Agent Instructions）内で以下のプリファレンスを調整可能：

- **1日あたりの最大会議時間** — 会議過負荷の閾値を設定
- **フォーカスタイム** — 集中作業用の確保時間を指定
- **優先参加者・会議** — 重要な参加者や会議のプライオリティを定義

## イントロダクション

Anthropicは **Project Glasswing**（名称はガラス翼の蝶 *Greta oto* に由来）を発表した。これは **AWS、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Linux Foundation、Microsoft、NVIDIA、Palo Alto Networks** の12社が参加し、世界の最も重要なソフトウェアのセキュリティ確保を目指すイニシアティブである。

背景にあるのは、Anthropicが訓練した未公開フロンティアモデル **Claude Mythos Preview** の能力である。このモデルは、ソフトウェア脆弱性の発見・悪用において、最も熟練した人間のセキュリティ専門家に匹敵、もしくは凌駕するレベルに到達している。

### 主要なコミットメント

- パートナー企業は Mythos Preview を防御的セキュリティ業務に活用
- 重要なソフトウェアインフラを構築・保守する **40以上の追加組織**にもアクセスを拡大
- Anthropicは **最大1億ドルの利用クレジット** を提供
- オープンソースセキュリティ組織への **400万ドルの直接寄付**（Alpha-Omega/OpenSSFに250万ドル、Apache Software Foundationに150万ドル）

## AI時代のサイバーセキュリティ

### 現状の脅威

日常的に依存しているソフトウェア（銀行システム、医療記録、物流ネットワーク、電力グリッドなど）には常にバグが存在し、一部は深刻なセキュリティ上の欠陥となっている。これまでの主なサイバー攻撃事例：

- 企業ネットワーク（Oracle E-Business Suite ゼロデイ）
- 医療システム（WannaCry / NHS）
- エネルギーインフラ（Colonial Pipeline）
- 交通拠点（欧州空港ランサムウェア）
- 政府機関（SolarWinds、米国財務省、英国政府）
- 病院・学校への個別攻撃

> サイバー犯罪の世界的なコストは **年間約5,000億ドル** と推定されている。

### AIによるパラダイムシフト

過去1年間で、AIモデルはコードの読解・推論能力が飛躍的に向上し、脆弱性発見とエクスプロイト開発の**コスト・労力・必要な専門知識**がすべて劇的に低下した。これは攻撃者にとっても防御者にとっても大きな変化である。

> **楽観的な見方**: AIモデルの能力は、悪用されるリスクがある一方で、重要ソフトウェアの欠陥を発見・修正し、セキュリティバグの少ない新しいソフトウェアを生産するために不可欠なツールとなり得る。

## Claude Mythos Preview の脆弱性発見能力

### 発見された脆弱性の具体例

過去数週間で、Mythos Preview はすべての主要OSおよび主要ウェブブラウザにおいて**数千のゼロデイ脆弱性**（ソフトウェア開発者に未知の欠陥）を特定した。ほぼすべてが**完全自律的**に、人間の指示なく発見された。

| 対象 | 脆弱性の詳細 | 影響 |
|------|-------------|------|
| **OpenBSD** | **27年間**存在していた脆弱性 | 接続するだけでリモートからマシンをクラッシュ可能。世界で最もセキュリティが強化されたOSの一つ |
| **FFmpeg** | **16年間**存在していた脆弱性。自動テストツールが該当コード行を**500万回**テストしても発見できなかった | 無数のソフトウェアで使用されるビデオエンコード/デコードライブラリ |
| **Linuxカーネル** | 複数の脆弱性を自律的に発見し**チェーン化** | 通常のユーザーアクセスからマシンの完全制御への権限昇格が可能 |

上記の脆弱性はすべて関係するメンテナーに報告され、すでにパッチが適用されている。

## Project Glasswing の今後の計画

### アクセスと価格

- パートナー企業は Mythos Preview で脆弱性の発見・修正を実施
- 想定される作業: ローカル脆弱性検出、バイナリのブラックボックステスト、エンドポイント保護、ペネトレーションテスト
- **1億ドルのモデル利用クレジット**で研究プレビュー中の大規模利用をカバー
- その後の価格: 入力 **$25** / 出力 **$125** (100万トークンあたり)
- 利用プラットフォーム: Claude API、Amazon Bedrock、Google Cloud Vertex AI、Microsoft Foundry

### 情報共有と提言

- **90日以内**に、学んだこと・修正した脆弱性・改善点を公開報告
- パートナー間での情報・ベストプラクティスの共有
- セキュリティ組織と連携し、AI時代のセキュリティ実践に関する**実践的推奨事項**を策定

### 推奨事項の対象分野

- 脆弱性開示プロセス
- ソフトウェア更新プロセス
- オープンソースおよびサプライチェーンセキュリティ
- ソフトウェア開発ライフサイクルとセキュア・バイ・デザイン
- 規制産業向け標準
- トリアージのスケーリングと自動化
- パッチ適用の自動化

### 一般公開について

Mythos Preview の**一般公開は予定されていない**。最終的な目標は、Mythosクラスのモデルを安全に大規模展開できるようにすること。そのために、モデルの最も危険な出力を検出・ブロックするサイバーセキュリティ・セーフガードの開発が必要。今後の **Claude Opus モデル**で新しいセーフガードをリリースし、改善・洗練を図る予定。

> セキュリティ専門家の正当な業務がセーフガードの影響を受ける場合、今後の **Cyber Verification Program** への申請が可能。

### 政府との連携

Anthropicは米国政府関係者と Mythos Preview の攻撃的・防御的サイバー能力について継続的に協議を行っている。重要インフラの保護は民主主義国家にとって最優先の国家安全保障上の課題であり、米国とその同盟国がAI技術で決定的なリードを維持する必要性を強調している。

## 重要な結論

1. **AIサイバー能力の閾値突破**: AIモデルは、最も熟練した人間のセキュリティ専門家に匹敵する脆弱性発見・悪用能力に到達した
2. **防御者の優位性構築**: 同じAI能力を防御的に活用することで、サイバー防御者に持続的な優位性を与えることが可能
3. **緊急性**: フロンティアAI能力は数ヶ月で大幅に進歩する見込みであり、防御側が先手を打つには今すぐ行動する必要がある
4. **業界横断の協力**: 単一の組織では解決できない問題であり、AI開発者・ソフトウェア企業・セキュリティ研究者・OSSメンテナー・政府すべてが不可欠な役割を担う
5. **オープンソースへの投資**: 世界の重要インフラを支えるOSSメンテナーへのセキュリティツール提供が、セキュリティの民主化に寄与する
